# DS03 - 卡牌驱动的探索 (Card-Driven Exploration)

**核心理念**: 融合《苏丹的游戏》卡牌机制 - 每个选择都有代价，没有"正确"答案

## 与DS01的核心差异

| 特性 | DS01 | DS03 |
|------|------|------|
| 操作方式 | 直接点击格子 | 打出卡牌执行行动 |
| 资源系统 | 理智+负重 | 行动点(AP)+理智 |
| 决策深度 | 即时的扫雷决策 | 策略性的卡牌管理 |
| 随机性 | 格子随机 | 抽牌随机+效果随机 |

## 核心机制

### 1. 行动点(AP)系统
- 每回合有3点行动点
- 打出卡牌消耗AP
- 结束回合恢复AP并抽牌

### 2. 手牌管理
- 手牌上限5张
- 牌库抽完会将弃牌堆洗牌
- 需要平衡"现在使用"vs"留着以后用"

### 3. 卡牌类型

**探索牌**
- 🚶 谨慎步伐 (⚡1) - 安全揭示1格
- ⚡ 鲁莽冲锋 (⚡1 🧠5) - 揭示3x3但损失理智
- 🔮 神启扫描 (⚡2) - 揭示多个安全格子

**工具牌**
- 🚩 标记陷阱 (⚡1) - 放置/移除标记
- 📍 探测杆 (⚡1) - 检查相邻格子安全性

**抉择牌** (苏丹风格核心)
- 🤝 黑暗交易 (⚡0) - 获得2张牌，失去10理智
- 🔪 血祭仪式 (⚡0) - 丢弃1物品，揭示5格
- 🎲 绝望赌博 (⚡1) - 50%大成功/50%大失败

**特殊牌**
- 💨 第二wind (⚡0 🧠10) - 恢复全部AP
- 🛡️ 心灵护盾 (⚡1) - 本回合免疫理智损失

### 4. 策略维度

资源权衡：
- 现在用牌还是留到更危险时？
- 消耗理智抽牌，还是保留理智保底？
- 献祭物品换取信息，还是保留物品卖钱？

风险控制：
- 稳妥的"谨慎步伐" vs 高回报的"鲁莽冲锋"
- 确定的代价 vs 赌博的不确定性

## 设计意图

DS03探索"卡牌驱动"的核心体验：
1. **延迟满足** - 好牌现在用还是以后用？
2. **代价意识** - 每个行动都有明确标价
3. **资源转换** - 用AP换信息，用理智换卡牌，用物品换视野
4. **道德困境** - 黑暗交易是否值得？

## 技术实现

- 完整的卡牌系统（牌库/手牌/弃牌堆）
- 多阶段结算（选择卡牌→选择目标→执行效果）
- 动态资源UI（行动点可视化）
