# DS04 - 双层结构 (Double Layer)

**核心理念**: 融合《苏丹的游戏》政治博弈 + 《黑暗之魂》的压抑氛围

## 架构设计

### 上层：深渊议会（政治层）

**四大派系：**

| 派系 | 图标 | 核心诉求 | 好感收益 |
|------|------|----------|----------|
| 商人公会 | 💰 | 收购深渊物品 | 售价+20% |
| 疯狂教会 | 😈 | 渴求诅咒物品 | 诅咒抗性 |
| 探险家协会 | 🗺️ | 探索与知识 | 额外休息次数 |
| 封印守护者 | 🔒 | 阻止深渊扩张 | 陷阱预警 |

**政治机制：**
- 派系好感度影响探索时的被动效果
- 必须平衡各方，过度偏向一方会触怒其他派系
- 随机政治事件需要做出抉择

### 下层：深渊探索（玩法层）

继承DS01的扫雷核心，但加入派系影响：

**派系增益在探索中的体现：**
- 商人好感>2：出售价格+20%
- 教会好感>3：诅咒物品不损失理智
- 探险家好感>5：额外1次休息
- 封印者好感>3：部分陷阱格子有红色边框预警

**派系惩罚：**
- 封印者好感<0：触发陷阱时额外损失理智（"守护者的冷眼"）

### 任务系统

- 每个派系发布专属委托
- 完成任务提升对应派系好感
- 部分任务相互冲突（如商人要化石，教会要神像）

## 核心体验

**选择的重量：**
- 把物品卖给商人还是献给教会？
- 帮助探险家协会分享地图（损失地图记忆）还是保密（-1好感）？
- 亵渎深渊获得力量（+教会 -封印者）还是保持敬畏？

**长期策略：**
- 专注讨好一个派系获得强力增益？
- 还是平衡各方避免被孤立？
- 不同的派系路线带来完全不同的游戏体验

## 技术实现

- 上层使用议会界面展示派系关系
- 下层保持扫雷核心玩法
- 两层通过派系数据相互影响
- 任务系统跟踪玩家行为
