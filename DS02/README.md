# DS02 - 死亡即进度 (Death is Progress)

**核心理念**: 融合《黑暗之魂》的死亡哲学 - 死亡不是失败，而是学习的一部分

## 与DS01的核心差异

| 特性 | DS01 | DS02 |
|------|------|------|
| 死亡后果 | 丢失本局物品，回酒馆 | 保留地图记忆，死亡地点生成墓碑 |
| 进度系统 | 成功撤离才保留物品 | 死亡本身积累"认知"，解锁新内容 |
| 地图机制 | 每局全新随机 | 永久保留已揭示的格子（跨局） |
| 特殊地点 | 撤离点 | 死亡墓碑（可掠夺死者遗物） |

## 核心机制

### 1. 死亡记忆系统
- 每次死亡的位置会生成**墓碑**（⚱️）
- 墓碑可以被后来的自己**掠夺**，获得死者遗物
- 亵渎墓碑会损失理智，但能获得稀有资源

### 2. 地图记忆保留
- 死亡后，已揭示的格子会**永久保留**记忆
- 下次进入同一层级，这些格子会显示为半透明（记忆痕迹）
- 减少重复探索的枯燥感

### 3. 死亡认知解锁
累计死亡次数解锁古神知识：
- 1次：你意识到痛苦会留下痕迹
- 3次：你开始听见死者的低语
- 5次：死亡地点会长出特殊的东西
- 10次：你分不清自己是生者还是死者
- 20次：深渊开始记住你的样子

### 4. 死亡奖励
- 初始理智上限随死亡次数增加（最高120）
- 死亡越多，对深渊的理解越深

## 新增物品

| 物品 | 效果 | 来源 |
|------|------|------|
| ⚱️ 灰烬余魂 | 任务道具，记录死亡记忆 | 墓碑掠夺 |
| 🌿 墓碑苔藓 | 普通道具，价值15 | 墓碑周围生长 |
| 🔮 回声石 | 揭示周围区域 | 深层墓碑稀有掉落 |

## 设计意图

DS02探索"魂like"的核心体验：
1. **死亡不是终点** - 每次死亡让世界更有意义
2. **重复中的变化** - 地图记忆减少重复劳动，墓碑增加新变数
3. **道德选择** - 亵渎自己的尸体获取资源，还是保持尊严？

## 技术实现

- 使用 localStorage 跨局保存死亡标记
- 使用 Set 结构高效存储已揭示格子
- 墓碑系统复用原有物品系统
