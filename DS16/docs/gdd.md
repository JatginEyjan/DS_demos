# DS16 - 沉默公约 (The Silence Convention)

**类型**: 反乌托邦模拟经营 / 时间管理 / 道德抉择  
**平台**: Web (H5)  
**核心体验**: 在"绝对静音即法律"的世界里，经营地下排气会所，在生存与良知间走钢丝。

---

## 1. 世界观设定

### 1.1 背景
2088年，"新静默纪元"第47年。《全球静音法案》已颁布近半个世纪：
- 任何超过20分贝的声响即构成"噪音犯罪"
- 放屁 = 一级重罪（死刑）
- 笑声 = 二级犯罪（终身监禁）
- 甚至咳嗽都要提前申请许可证

城市被**声呐监控网格**覆盖，AI"聆听者"无处不在。

### 1.2 你的角色
你是"排气会所"（The Vent）的经营者，代号：**风阀**。
- 表面上：废弃澡堂/地下洗衣房/退休图书馆管理员
- 实际上：城市唯一能让人们安全"释放"的地方
- 客户：从擦窗工人到市政高层，人人都憋着呢

## 2. 核心玩法

### 2.1 日循环机制
每天分为 **6个时段**（每时段约现实2分钟）：
```
晨间(06-10) | 正午(10-14) | 午后(14-18) | 黄昏(18-22) | 深夜(22-02) | 凌晨(02-06)
```

每个时段你决定：
1. **接谁？** — 客户预约列表（超出容量会排到下时段）
2. **用什么房？** — 普通/隔音/VIP（成本递增）
3. **风险？** — 该时段巡逻密度、告密者概率

### 2.2 客户系统

**客户三属性：**
- **憋压值** (0-100): 满100客户会"失控释放"（暴露会所）
- **社会等级**: 平民/中产/精英/VIP
- **忍耐力**: 等多久会愤怒离开

**客户状态机：**
```
预约 -> 等待中 -> 入房释放 -> 满意离开
           ↓
      憋压满/等太久 -> 失控暴露 -> 游戏结束
```

### 2.3 房间系统

| 房间类型 | 隔音等级 | 建设成本 | 维护/日 | 特殊效果 |
|---------|---------|---------|--------|---------|
| 普通隔间 | ★☆☆☆☆ | 免费（开局1间） | 低 | 无 |
| 吸音密室 | ★★★☆☆ | 500信用点 | 中 | 降低50%暴露风险 |
| 真空舱 | ★★★★☆ | 1500信用点 | 高 | 几乎无风险，但客户会缺氧（限时） |
| VIP套房 | ★★★★★ | 4000信用点 | 极高 | 可接待最高级客户，但引人注目 |

**风险事件：**
- **随机巡查**: 时段开始时概率触发，检查随机房间
- **告密者**: 客户中有概率混入驻军卧底，会记录证据
- **设备故障**: 隔音层老化暴露

### 2.4 资源管理

**信用点** — 主要货币：
- 收入：客户付费（VIP给得多，平民给得少）
- 支出：房间维护、封口费、升级设备

**声望** — 隐藏数值（-100~+100）：
- +：善待平民、拒绝权贵插队、口碑传播
- -：频繁让VIP插队、导致平民憋炸、被举报
- 影响：客户质量、卧底出现率、最终结局

**热量** — 关注度（0-100）：
- 操作越张扬（VIP多、暴露风险高）热量上升
- 热量高 = 巡查频率增加、客户不敢来

## 3. 道德抉择系统

### 3.1 典型困境

**【VIP插队】**
- 市政官的秘书来电：秘书长要在正午时段插队，但他前面还有3个憋压90+的平民
- 选项A：拒绝 — 失去VIP客户，但保全3个平民
- 选项B：接受 — 秘书长给巨额小费，但3个平民中必有一个憋炸暴露

**【告密者嫌疑】**
- 新客户行为异常（东张西望、不问价、直接要最好房间）
- 选项A：正常接待 — 可能是卧底，收集证据后你会被捕
- 选项B：拒绝服务 — 如果不是卧底，差评传播，声望下降
- 选项C："意外" — 安排他进故障房间...（永久解决，但良心？）

**【资源短缺】**
- 月底钱不够维护所有房间
- 选项A：关闭平民区 — 他们无处可去，街头"爆炸"新闻+1
- 选项B：拖欠VIP房维护 — VIP暴露风险大增，热量飙升
- 选项C：借高利贷 — 下月压力更大，陷入死亡螺旋

### 3.2 结局分支

| 结局 | 触发条件 | 描述 |
|-----|---------|------|
| **殉道者** | 被捕，声望极高 | 你的审判成为反抗运动导火索 |
| **权贵走狗** | 存活，声望极低 | 你成了精英们的专属厕所，良心已死 |
| **地下传奇** | 存活100天，热量<30 | 你建立了一套完美的地下秩序 |
| **大爆炸** | 让5个客户憋炸 | 城市因"连环屁爆"陷入混乱 |
| **改革者** | 特殊事件链完成 | 你收集证据推动了法案修订（真结局） |

## 8. 补充机制设计

### 8.1 屁的处理系统 (Gas Disposal)

客户释放后产生的"产物"必须处理，否则积累暴露：

| 处理方式 | 成本 | 风险/副作用 | 适用场景 |
|---------|------|------------|---------|
| **直排** | 免费 | 热量+10/次，有概率触发监测警报 | 平民客户的"轻量"排气 |
| **活性炭过滤** | 50信用点/次 | 设备会饱和需更换，无热量增加 | VIP客户（硫化物含量高） |
| **储存罐** | 200信用点/罐 | 罐满爆炸=游戏结束；可后期黑市出售 | 特殊客户的高浓度样本 |

**策略深度**：VIP的屁"价值高"（含信息素/药物残留）必须过滤处理；平民的屁可以偷偷直排省钱，但积累风险。

### 8.2 获客与产能系统 (Acquisition & Yield)

**客户来源三渠道**：
- **口碑传播**（免费但慢）：服务好 → 客户告诉朋友 → 解锁新客户类型
- **地下营销**（有风险）：雇人在街头贴小广告 → 获客+20% → 但热量+15
- **食物供应链**（长期布局）：和地下食堂合作，送产气食物（豆类、洋葱）→ 客户"产能"翻倍 → 但处理成本+50%

**客户产能管理**：
- 普通客户：1次/天
- 吃了豆子的客户：3次/天（但气味等级+2，必须用活性炭）

### 8.3 房间拓展机制 (Expansion)

不是简单的"购买"，受物理空间限制：

```
当前空间容量: ████████░░ 80%已满
```

**拓展方式**：
| 方式 | 条件 | 成本 | 风险 |
|-----|------|------|------|
| **横向挖隧** | 贿赂城市规划局 | 2000信用点+声望-10 | 可能挖到监听设备或其他地下组织地盘 |
| **纵向深挖** | 加固支撑结构 | 指数增长（第2层×2，第3层×4）| 结构不稳会导致塌方 |
| **时间共享** | 无 | 免费 | 同一房间分时段使用，增加客户等待冲突概率 |

### 8.4 释放耗时机制 (Duration)

每个客户有"释放耗时"属性，影响房间周转率：

| 客户类型 | 耗时 | 特点 |
|---------|------|------|
| 急性子商人 | 30秒 | 给双倍小费，但不能等 |
| 谨慎老干部 | 3分钟 | 耗时久但几乎无声（风险-30%）|
| 便秘患者 | 5分钟+ | 可能需要"辅助药物"（额外成本100信用点）|
| 紧张新手 | 1-10分钟随机 | 首次客户，时间不确定 |

**排班策略**：把快客户排一起（流水线）；慢客户独占房间但占用资源久。

### 8.5 憋压危险状态 (Pressure States)

客户等待时憋压持续上升，三级危险：

```
憋压 0-70:   🟢 安全区（正常等待）
憋压 70-90:  🟡 危险区（客户出汗、坐立不安）
憋压 90-99:  🔴 临界区（身体颤抖，随时可能...）
憋压 100:    💥 失控释放 → 房间暴露 → 游戏结束
```

**特殊事件：憋死但没暴露**
- 客户在你这里憋死了（物理死亡）
- 必须处理尸体：
  - A. 秘密埋葬（500信用点，风险：被发现=游戏结束）
  - B. 卖给黑市医院（200信用点收入，道德-30）
  - C. 伪装成自然死亡（需要贿赂法医，800信用点）

### 8.6 反举报防御系统 (Anti-Report Defense)

**告密者识别**：
- 新客户首次来：标记"可疑"
- 行为分析：
  - 问太多问题（"你们安全措施怎么样？"）= 概率卧底
  - 直接脱裤子办事 = 真客户

**反制手段层级**：

| 手段 | 成本 | 道德影响 | 效果 |
|-----|------|---------|------|
| **预付封口费** | 100信用点 | 中性 | 降低50%举报概率 |
| **建立眼线网** | 长期维护200/天 | +10道德 | 忠诚客户提前警告卧底信息 |
| **安排故障房** | 免费 | -20道德 | 如果是卧底，"意外"死亡 |
| **掌握把柄** | 任务解锁 | -30道德 | 用客户秘密要挟不敢举报 |

**声望连锁**：频繁使用"故障房"→消息传开→真客户不敢来→进入恶性循环。
