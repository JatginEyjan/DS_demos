# DS16 - 沉默公约 (The Silence Convention)

**类型**: 反乌托邦模拟经营 / 时间管理 / 道德抉择  
**平台**: Web (H5)  
**核心体验**: 在"绝对静音即法律"的世界里，经营地下排气会所，在生存与良知间走钢丝。

---

## 1. 世界观设定

### 1.1 背景
2088年，"新静默纪元"第47年。《全球静音法案》已颁布近半个世纪：
- 任何超过20分贝的声响即构成"噪音犯罪"
- 放屁 = 一级重罪（死刑）
- 笑声 = 二级犯罪（终身监禁）
- 甚至咳嗽都要提前申请许可证

城市被**声呐监控网格**覆盖，AI"聆听者"无处不在。

### 1.2 你的角色
你是"排气会所"（The Vent）的经营者，代号：**风阀**。
- 表面上：废弃澡堂/地下洗衣房/退休图书馆管理员
- 实际上：城市唯一能让人们安全"释放"的地方
- 客户：从擦窗工人到市政高层，人人都憋着呢

## 2. 核心玩法

### 2.1 日循环机制
每天分为 **6个时段**（每时段约现实2分钟）：
```
晨间(06-10) | 正午(10-14) | 午后(14-18) | 黄昏(18-22) | 深夜(22-02) | 凌晨(02-06)
```

每个时段你决定：
1. **接谁？** — 客户预约列表（超出容量会排到下时段）
2. **用什么房？** — 普通/隔音/VIP（成本递增）
3. **风险？** — 该时段巡逻密度、告密者概率

### 2.2 客户系统

**客户三属性：**
- **憋压值** (0-100): 满100客户会"失控释放"（暴露会所）
- **社会等级**: 平民/中产/精英/VIP
- **忍耐力**: 等多久会愤怒离开

**客户状态机：**
```
预约 -> 等待中 -> 入房释放 -> 满意离开
           ↓
      憋压满/等太久 -> 失控暴露 -> 游戏结束
```

### 2.3 房间系统

| 房间类型 | 隔音等级 | 建设成本 | 维护/日 | 特殊效果 |
|---------|---------|---------|--------|---------|
| 普通隔间 | ★☆☆☆☆ | 免费（开局1间） | 低 | 无 |
| 吸音密室 | ★★★☆☆ | 500信用点 | 中 | 降低50%暴露风险 |
| 真空舱 | ★★★★☆ | 1500信用点 | 高 | 几乎无风险，但客户会缺氧（限时） |
| VIP套房 | ★★★★★ | 4000信用点 | 极高 | 可接待最高级客户，但引人注目 |

**风险事件：**
- **随机巡查**: 时段开始时概率触发，检查随机房间
- **告密者**: 客户中有概率混入驻军卧底，会记录证据
- **设备故障**: 隔音层老化暴露

### 2.4 资源管理

**信用点** — 主要货币：
- 收入：客户付费（VIP给得多，平民给得少）
- 支出：房间维护、封口费、升级设备

**声望** — 隐藏数值（-100~+100）：
- +：善待平民、拒绝权贵插队、口碑传播
- -：频繁让VIP插队、导致平民憋炸、被举报
- 影响：客户质量、卧底出现率、最终结局

**热量** — 关注度（0-100）：
- 操作越张扬（VIP多、暴露风险高）热量上升
- 热量高 = 巡查频率增加、客户不敢来

## 3. 道德抉择系统

### 3.1 典型困境

**【VIP插队】**
- 市政官的秘书来电：秘书长要在正午时段插队，但他前面还有3个憋压90+的平民
- 选项A：拒绝 — 失去VIP客户，但保全3个平民
- 选项B：接受 — 秘书长给巨额小费，但3个平民中必有一个憋炸暴露

**【告密者嫌疑】**
- 新客户行为异常（东张西望、不问价、直接要最好房间）
- 选项A：正常接待 — 可能是卧底，收集证据后你会被捕
- 选项B：拒绝服务 — 如果不是卧底，差评传播，声望下降
- 选项C："意外" — 安排他进故障房间...（永久解决，但良心？）

**【资源短缺】**
- 月底钱不够维护所有房间
- 选项A：关闭平民区 — 他们无处可去，街头"爆炸"新闻+1
- 选项B：拖欠VIP房维护 — VIP暴露风险大增，热量飙升
- 选项C：借高利贷 — 下月压力更大，陷入死亡螺旋

### 3.2 结局分支

| 结局 | 触发条件 | 描述 |
|-----|---------|------|
| **殉道者** | 被捕，声望极高 | 你的审判成为反抗运动导火索 |
| **权贵走狗** | 存活，声望极低 | 你成了精英们的专属厕所，良心已死 |
| **地下传奇** | 存活100天，热量<30 | 你建立了一套完美的地下秩序 |
| **大爆炸** | 让5个客户憋炸 | 城市因"连环屁爆"陷入混乱 |
| **改革者** | 特殊事件链完成 | 你收集证据推动了法案修订（真结局） |

## 4. 美术风格

- **基调**: 赛博朋克 + 苏联式官僚压抑
- **色调**: 灰蓝（监控网格）、暗红（危险/憋压）、暖黄（会所内的安全感）
- **UI**: 复古终端风格，绿色CRT扫描线效果
- **角色**: 像素画，客户戴口罩（压抑），只有在你会所里才能摘下来松口气

---

## 5. 技术栈

- **引擎**: Phaser 3 (沿用DS15基础)
- **UI**: DOM overlay for 管理界面
- **音频**: 极简（ ironic：关于声音的游戏却几乎静音，只在"释放"时有低频震动反馈）

## 7. 里程碑

- **MVP** (3-5天): 基础日循环、1种房间、3类客户、被捕结局
- **Alpha** (1-2周): 多房间类型、完整客户系统、2个道德事件
- **Beta** (2-3周): 全部结局、美术完整、音频反馈
- **Release** (1周): 平衡调整、多周目继承（真结局解锁）
