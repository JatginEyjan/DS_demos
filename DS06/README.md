# DS06 - 碎片化叙事 (Fragmented Narrative)

**核心理念**: 融合《伊迪芬奇的记忆》《黑暗之魂》的环境叙事

## 核心机制

### 每个格子都有故事

不同于传统扫雷，DS06中**15%的格子**都隐藏着叙事片段：
- 墙壁上的涂鸦
- 散落的日记
- 诡异的照片
- 不可名状的痕迹

### 叙事收集系统

**记忆之家（酒馆）**
- 查看已收集的所有故事
- 真相收集进度条
- 4个真结局碎片需要集齐才能理解完整真相

**叙事类型**
- 📖 **记忆** - 过去探险者的经历
- ⚠️ **警告** - 对后来者的告诫
- 💀 **恐怖** - 不可名状的遭遇
- 📜 **诗歌** - 疯狂的隐喻
- 🗝️ **真相碎片** - 解锁真结局的线索

### 真结局系统

收集全部4个真相碎片：
- 【碎片A】"深渊不是地下，是另一个维度"
- 【碎片B】"他们不是死了，是进去了"
- 【碎片C】"每次"撤离"就有一个它跟着上去"
- 【碎片D】"是深渊在通过你的眼睛看世界"

## 设计意图

**探索驱动**
- 扫雷不只是数学谜题，更是发现故事的过程
- 每个揭示的格子都可能带来惊喜/惊吓

**多周目价值**
- 不同层级有不同叙事
- 深层需要死亡后才能解锁（暗示前作DS02机制）

**氛围营造**
- 使用衬线字体营造阅读感
- 叙事面板常驻右侧，强化"阅读体验"
- ✨标记吸引玩家探索

## 技术亮点

- 完整的叙事片段库系统
- 真结局碎片随机位置生成
- 物品也附带故事（点击背包物品可阅读）
