# 游戏思路文档

> **项目代号**：DS18
> **类型**：2D H5 卡牌编排式 Roguelike
> **版本**：v0.1（概念阶段）
> **最后更新**：2026-04-02

---

## 一、核心概念

### 一句话描述

魂系风格的卡牌编排式 Roguelike——玩家作为"命运编排者"，用卡牌构建勇者每天的冒险时间轴，看他在黑暗世界中自动战斗、成长，最终挑战 Boss。

### 核心幻想

> 你不是勇者，你是勇者背后的命运之手。

### 参考坐标系

| 游戏 | 借鉴维度 |
|------|---------|
| Loop Hero | 卡牌编排决定世界运转 |
| Vampire Survivors | 单局节奏（20~30 分钟）、permadeath 紧张感 |
| Dark Souls 3 / Elden Ring | 角色成长体系、灵魂经济、死亡机制、氛围 |
| Slay the Spire | Roguelike 结构、三选一、路线选择 |
| Elden Ring: Night Reign | 多日循环、Mini-Boss 节点、战役路线 |

---

## 二、核心循环

```
局外选择（职业 / 遗物 / 战役路线）
    └─▶ 局内循环（根据路线 4~8 天，每天 3~5 分钟，总计 20~30 分钟）
          ├─ 规划阶段（不限时）：安排卡槽、管理背包、编辑未来天
          ├─ 执行阶段（自走）：勇者按时间轴推进，触发事件，玩家做微决策
          ├─ 结算阶段：精英战 / 三选一 / 篝火升级
          └─ Boss 逼近度 ▲
    └─▶ Boss 战（多阶段策略）
    └─▶ 结算 / 局外进度更新
```

## 三、卡槽与时间轴

### 基础规则

- 每天的可用时长 = **耐力属性值**（初始约 12h，可升级到 20h+）
- 每张卡有固定时长（2h / 3h / 5h / 8h…）
- 玩家依次放入卡牌，累计时长推进时钟
- 前半段 = 白天，后半段 = 夜晚（按当天总耐力的 50% 分界）
- 未填满的时间 = 自动休息（少量 HP 恢复，无收益）
- **返回卡**可放任意位置，提前结束当天

### 白天 / 夜晚

- 白天：怪物较弱、收益较低、信息更清晰
- 夜晚：怪物更强、收益更高、信息更模糊
- 玩家可通过调整卡牌顺序决定哪些卡落在白天 / 夜晚区间
- 存在时间操控卡（日蚀 / 黎明 / 篝火 / 加速）可主动操控日夜

### "不填满"是策略

| 策略 | 效果 |
|------|------|
| 填满全部耐力 | 收益最大化，但消耗重、风险高 |
| 只填部分 | 留空休息回血，保守但稳健 |
| 故意安排夜晚卡 | 触发夜晚特殊效果，高风险高回报 |

## 四、卡牌系统

### 三大卡组

| 大类 | 包含 | 使用场景 |
|------|------|---------|
| **路程卡** | 房间卡、地形卡、时间操控卡、返回卡 | 放入卡槽构建时间轴 |
| **镶嵌卡** | NPC 卡、事件卡 | 镶嵌到房间卡内部 |
| **角色卡** | 装备卡、战灰卡、消耗品卡 | 装备到勇者 / 背包中使用 |

### 卡牌分层（渐进解锁）

```
前期（Day 1~2）:  [房间卡]
中期（Day 3~5）:  [房间卡] → 镶嵌 [NPC×1] + [事件×1]
后期（Day 6+） :  [地形卡] → [房间卡] → 镶嵌 [NPC×1~2] + [事件×1~2]
```

房间卡的品质决定可镶嵌的 NPC / 事件卡的品质上限。

### 永久 / 消耗分类

| 卡牌类型 | 永久版 | 消耗版 |
|---------|--------|--------|
| 房间卡 | 有（核心构筑，有探索度机制） | 有（强力单次，满探索度体验） |
| NPC 卡 | 有（使用后冷却 1 天） | 有（效果更强，用完消失） |
| 事件卡 | **无** | **只有消耗** |
| 地形卡 | 有 | 有（临时地形改造） |
| 装备卡 | 有（穿戴） | - |
| 消耗品卡 | - | 有（药水 / 炸弹） |
| 战灰卡 | 有（镶嵌武器） | - |
| 时间操控卡 | 有 | 有 |

### 房间探索度

- 第 1 次进入：基础内容
- 第 2 次进入：解锁隐藏区域（更深处、更强的怪、更好的掉落）
- 第 3 次进入：触发特殊事件（一次性）
- 之后：效果衰减，鼓励轮换

### 卡牌 Combo

- 相邻卡槽加成（不死聚落 + 墓地 = 不死系掉落翻倍）
- 地形连锁（连续相同地形触发共鸣）
- NPC 组合（铁匠 + 矿洞 = 直接打造装备）
- 日夜跨越奖励（横跨分界线的卡有特殊效果）

## 五、跨天预布局

### 核心概念

执行当天的同时，玩家可以用**已拥有**的卡牌编辑未来天的卡槽。

### 规则

- 放入卡牌：免费
- 取回已放置的卡牌：需花费灵魂（承诺成本）
  - Day 1~2 的卡槽编辑：免费取回（新手保护期）
  - Day 3+：取回费 = 卡牌品质 × 基础系数（普通 10 / 稀有 30 / 史诗 80 / 传说 200 灵魂）
- 当天开始执行后，该天卡槽**锁定不可更改**

### 策略价值

- 提前规划完美的一天 vs 留灵魂升级
- 自走阶段"安静段"自然切到规划，"紧张段"拉回当下
- 长线策略（Day 1 规划 Day 5 的 Combo）vs 短线应急

## 六、魂系角色成长

### 灵魂——统一货币

- 击杀怪物、完成事件、骰子奖励 → 获得灵魂
- 灵魂同时用于**升级** AND **消费**（买卡 / NPC 交易 / 取回预布局卡）
- 核心张力：存着升级 还是 花掉买卡？

### 六维属性

| 属性 | 战斗效果 | 系统效果 |
|------|---------|---------|
| 生命力 | 最大 HP、自然恢复 | - |
| 耐力 | - | 每天可用卡槽总时长 |
| 力量 | 物理伤害 | 背包容量 |
| 敏捷 | 闪避率、暴击率、先手 | - |
| 智力 | 魔法伤害 | 骰子事件判定加成 |
| 信仰 | 治疗效果 | NPC 好感度加成 |

- 篝火升级，花费递增，有软上限
- **不可洗点**——选择不可撤回

### 武器 / 装备系统

- 武器卡有 S/A/B/C/D 补正等级（力量 / 敏捷 / 智力 / 信仰）
- 属性需求门槛
- 武器特技（自走阶段自动触发）
- 可镶嵌战灰卡改变补正和特技
- 强化系统：+1~+10（需强化材料 + 铁匠 NPC）

### 治疗瓶

- 每天初始化固定瓶数（初始 3 瓶）
- 通过事件 / NPC 永久 +1
- 信仰提升每瓶回复量
- 篝火卡刷新瓶数

### 四条 Build 路线

| Build | 风格 | 偏好卡牌 |
|-------|------|---------|
| 力量型 | 高伤高血，正面碾压 | 重武器、重甲、矿洞、锻造 |
| 敏捷型 | 高闪避暴击，战斗效率高 | 轻武器、陷阱、密道、刺客 NPC |
| 智力型 | 骰子加成，事件收益最大化 | 魔法、奥术塔、图书馆、法师 NPC |
| 信仰型 | 治疗强，NPC 交互收益高 | 神迹、祝福、教堂、牧师 NPC |

## 七、自走阶段玩家交互

| 交互类型 | 触发方式 | 示例 |
|----------|---------|------|
| DnD 骰子事件 | 事件卡触发 | "上锁的门——≥4 成功获宝物，＜4 触发陷阱" |
| 战利品选择 | 击杀怪物后 | 掉落 3 张卡选 1 张，或分解为资源 |
| NPC 交互 | NPC 卡触发 | 防火女：消耗灵魂升级 / 商人：买卡 |
| 紧急抉择 | 随机 / 条件触发 | "HP 低于 30%——是否使用应急药水？" |
| 背包整理 | 随时 | 整理背包、预排未来天卡组 |

设计原则：交互出现时**不暂停或短暂暂停**时间轴，超时使用默认选项。

## 八、死亡机制——魂系一命容错

| 状态 | 触发 | 效果 |
|------|------|------|
| 第一次死亡 | HP 归零 | 当天强制结束，未花费灵魂掉落在死亡卡槽，进入下一天 |
| 回收灵魂 | 下一天安排同一房间卡 | 回收掉落灵魂（致敬血迹回收） |
| 未回收再死 | 再次 HP 归零 | 掉落灵魂永久消失 + Game Over |
| 已回收再死 | HP 归零 | 重新进入"第一次死亡"状态 |

## 九、战役路线与 Boss 系统

### 路线结构

```
短线战役（入门）：4 天，无前置 Boss
中线战役（标准）：6 天，1 个 Mini-Boss
长线战役（挑战）：8 天，2 个 Mini-Boss
```

- 局开始前选定路线
- 不同路线对应不同最终 Boss、不同天数、不同 Mini-Boss 节点
- 路线选择界面展示：天数、Mini-Boss、最终 Boss 名称及弱点属性提示
- 通关后解锁新路线

### Mini-Boss

- 路线中固定节点，简化版 Boss 战（2 阶段）
- 奖励是**质变性**的（解锁新卡池 / 新武器 / 核心天赋）
- 失败遵循一命容错规则

### Boss 逼近度

```
每过一天        +10~15
跳过精英战      +5
使用强力贪心卡   +2
击败精英        -3
```

- 只在结算阶段检查，满 100 强制进入 Boss 战
- 70+ 可主动挑战（奖励更好）
- 满值强制战时 Boss 获得强化

### Boss 战设计

- 多阶段（Mini-Boss 2 阶段，最终 Boss 3~4 阶段）
- 阶段间选择策略（全力输出 / 防御回复 / 切换武器）
- Boss 大招有预警→选择应对
- 可使用储存的消耗卡
- Boss 弱点与 Build 相关

## 十、经济系统

### 获取来源

- 战斗掉落（灵魂 + 卡牌 + 材料）
- 骰子事件奖励
- NPC 交易
- 精英战 / Boss 三选一

### 使用方式

- 篝火升级属性
- NPC 处消费（买卡 / 升级 / 强化武器）
- 合成 / 升级卡牌
- 预布局取回费
- 分解卡牌为基础资源

### 背包管理

- 格子上限 15~20 张（力量属性扩容）
- 超出需丢弃或分解
- 特定 NPC / 事件可扩容

## 十一、信息可见性

| 层级 | 内容 | 时机 |
|------|------|------|
| 确定 | 房间基础怪物、NPC 功能、卡牌效果 | 规划时可见 |
| 模糊 | 预估难度 / 收益指标，夜晚比白天更模糊 | 规划时可见 |
| 未知 | 骰子事件结果、稀有掉落、随机遭遇 | 执行时揭示 |

---

## 十二、惊喜感维持

### 稀有遭遇系统

极低概率在某些房间触发完全意外的事件（隐藏 Boss、神秘商人、时空裂缝）。

### 每局诅咒 / 祝福

- 每局开始：随机 1 祝福 + 1 诅咒
- 祝福在局开始时展示，诅咒在 Day 2 才揭示
- 诅咒可通过特定 NPC / 事件移除（代价高）
- 好的诅咒不是纯负面，而是改变策略偏好

## 十三、局外 Meta 系统

| 元素 | 说明 |
|------|------|
| 职业选择 | 决定初始属性 / 初始卡牌 / 初始装备 |
| 遗物选择 | 局外永久解锁的被动加成 |
| 战役路线 | 选择目标 Boss，决定天数和 Mini-Boss 节点 |
| 解锁进度 | 通关不同路线解锁新职业 / 遗物 / 路线 |
