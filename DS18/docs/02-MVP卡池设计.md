# MVP 卡池设计文档

> **项目代号**：DS18
> **版本**：v0.1（MVP 原型阶段）
> **最后更新**：2026-04-02
> **目标**：设计最小可验证卡池，支撑一局完整的 3 天短线战役（Day1 → Day2 → Day3-Boss）

---

## 一、MVP 范围界定

### 验证目标

用最少的内容验证核心循环"**规划 → 自走 → 结算**"是否有趣。

### MVP 包含

- 1 条战役路线（3 天，无 Mini-Boss，直接打 Boss）
- 1 个初始职业（战士——力量 / 生命力偏向）
- 卡牌系统只开放前两层（房间卡 + 镶嵌 NPC / 事件）
- 不含地形卡层级
- 不含诅咒 / 祝福系统
- 不含预布局灵魂取回费（全免费，简化测试）

### MVP 卡池规模

| 类型 | 数量 | 说明 |
|------|------|------|
| 房间卡（永久） | 6 张 | 核心构筑骨架 |
| 房间卡（消耗） | 2 张 | 强力单次体验 |
| NPC 卡（永久） | 4 张 | 带冷却 |
| NPC 卡（消耗） | 2 张 | 用完消失，效果更强 |
| 事件卡（消耗） | 6 张 | 全部为消耗 |
| 装备卡 | 5 张 | 武器为主 |
| 消耗品卡 | 3 张 | 药水 / 炸弹 |
| 时间操控卡 | 2 张 | 篝火 + 返回 |
| **合计** | **30 张** | |

## 九、MVP Boss 设计

### 堕落骑士（最终 Boss）

**背景**：曾经是守护圣地的骑士，被深渊侵蚀堕落。

| 阶段 | HP 占比 | 行为 |
|------|--------|------|
| 第一阶段 | 100%~60% | 普通攻击 + 盾击（可被格挡） |
| 第二阶段 | 60%~30% | 攻速提升，新增"突刺连击"（3 连击） |
| 第三阶段 | 30%~0% | 深渊化，攻击附带腐蚀（持续伤害），新增"深渊风暴"大招 |

**大招预警与应对**：

| 大招 | 预警 | 玩家选择 |
|------|------|---------|
| 盾击 | 1 秒举盾动作 | 闪避（敏捷判定）/ 硬扛 / 使用消耗品 |
| 突刺连击 | 2 秒蓄力 | 防御姿态（减伤 50%）/ 全力输出（赌在蓄力期打出硬直） |
| 深渊风暴 | 3 秒聚能 | 闪避 / 使用火焰壶打断 / 硬扛（需 HP > 40%） |

**弱点**：火属性伤害 +30%（配合火焰壶 / 火焰附魔 / 战灰可利用）

**推荐通关属性（参考）**：

| 属性 | 最低建议值 |
|------|-----------|
| 生命力 | 16+ |
| 力量 | 18+（大剑流）或 敏捷 18+（弯刀流） |
| 武器强化 | +3 以上 |

## 十、MVP 一局流程模拟

### Day 1（起步）

**初始手牌**：废弃墓地、林间小路、宝箱陷阱、返回卡、篝火卡

**规划**：
```
[林间小路 2h] → [废弃墓地 3h（镶嵌：宝箱陷阱）] → [篝火 2h] → [返回 0h]
耐力使用：7h / 14h（留 7h 休息回血）
```

**执行**：
- 林间小路：击杀野狼 ×2，获得灵魂 20，掉落「绿花草」
- 废弃墓地：击杀骷髅兵 ×3，获得灵魂 40，触发宝箱陷阱（骰子投 5→成功，获得「灵魂漩涡」事件卡）
- 篝火：回复 HP，消耗灵魂 100 升级力量 12→13

**结算**：跳过精英战（太早了），Boss 逼近度 +10

**新获得**：灵魂 60（剩余）、绿花草 ×1、灵魂漩涡 ×1

### Day 2（成长）

**手牌新增**：（假设 Day 1 掉落了「商人 NPC」卡）

**规划**：
```
[废弃墓地 3h（镶嵌：灵魂漩涡）] → [林间小路 2h（镶嵌：商人）] → [新手村 3h] → [篝火 2h]
耐力使用：10h / 14h
```

**执行**：
- 废弃墓地（探索度 2）：解锁地下墓穴层，击杀精英骷髅，灵魂漩涡 ×2 效果，获得大量灵魂 + 普通石 ×1
- 林间小路（探索度 2）：发现猎人小屋，战斗后商人出现，用灵魂购买「大剑」
- 新手村：刷怪获取少量灵魂和金币
- 篝火：升级力量到 18，装备大剑

**结算**：挑战精英战，三选一选择「火焰壶 ×2」，Boss 逼近度 +10-3=+7

### Day 3（决战）

**规划**：
```
[矿洞 4h（镶嵌：铁匠）] → [古老教堂 3h] → [篝火 2h] → Boss 战
耐力使用：9h / 15h（升过耐力）
```

**执行**：
- 矿洞 + 铁匠：获取大块石，铁匠强化大剑到 +3
- 古老教堂（白天）：治疗瓶 +1（当天），祈祷回复
- 篝火：满血满瓶，最后升级生命力

**Boss 战**：
- 携带：火焰壶 ×2、绿花草 ×1、魔力树脂 ×1
- 第一阶段：正常输出，用魔力树脂
- 第二阶段：选防御姿态应对突刺连击
- 第三阶段：火焰壶打断深渊风暴 + 全力输出
