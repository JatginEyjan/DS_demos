## 1. 游戏概述

### 1.1 高概念

玩家扮演"命运编排者"，通过在时间轴卡槽中安排各类卡牌，规划勇者每一天的冒险行程。勇者会自动沿时间轴推进，经历战斗、遭遇 NPC、触发事件。玩家的核心操作是**策略规划**和**资源管理**，而非即时操控战斗。

### 1.2 核心卖点

1. **卡牌编排时间轴**——独特的核心玩法，用卡牌"编程"勇者的一天
2. **跨天预布局**——当天执行时可规划未来天的卡槽，带有灵魂承诺成本
3. **魂系成长融入 Roguelike 单局**——灵魂双用、不可洗点、装备补正
4. **魂系死亡容错**——一命容错 + 灵魂掉落回收机制

### 1.3 目标受众

- 喜欢策略 / 卡牌构筑的玩家（Slay the Spire、Loop Hero 受众）
- 喜欢魂系氛围但不擅长动作操作的玩家
- 碎片时间玩家（H5 平台，20~30 分钟单局）

### 1.4 核心体验目标

| 体验 | 来源 |
|------|------|
| "再来一局" | Roguelike 随机性 + 短局时长 |
| "这把我要试另一个 Build" | 四条属性路线 + 装备补正系统 |
| "刚好最优解！" | 卡牌 Combo + 预布局策略 |
| "紧张到手心出汗" | Boss 逼近度 + 一命容错的压力 |
| "这个氛围太棒了" | 魂系美术 + 音乐 + 叙事碎片 |

## 2. 核心循环

### 2.1 宏观循环

```
局外准备 → 局内 Day Loop × N → Boss 战 → 局结算 → 局外进度
```

### 2.2 单天循环

```
规划阶段（不限时）→ 执行阶段（实时自走）→ 结算阶段
```

### 2.3 规划阶段

**玩家操作**：
- 从背包中选取卡牌放入当天卡槽
- 安排卡牌顺序（影响白天 / 夜晚分配）
- 在房间卡中镶嵌 NPC / 事件卡
- 装备 / 更换武器和防具
- 查看模糊预测（预估难度 / 收益）
- 编辑未来天的卡槽（跨天预布局）

**约束条件**：
- 卡牌总时长不能超过当天耐力值
- 镶嵌受房间品质限制
- 装备受属性需求限制
- 背包受力量属性限制格子数

### 2.4 执行阶段

**自动运行**：
- 勇者按时间轴从左到右推进
- 战斗自动进行，武器特技自动触发
- 怪物按房间卡配置生成

**玩家交互**（不暂停或短暂暂停时间轴）：
- 骰子事件弹窗（限时决策，超时默认选项）
- 战利品选择（保留 / 分解）
- NPC 交互对话
- 治疗瓶手动 / 自动使用
- 消耗品使用
- 背包整理 + 未来天预布局

### 2.5 结算阶段

**可选内容**：
- 精英战 → 三选一奖励（属性 / 卡牌 / 消耗品）
- 阶段 Boss（路线节点）
- 跳过（直接进入下一天）

**固定内容**：
- 篝火升级（消耗灵魂升属性，花费递增）
- Boss 逼近度更新

（MVP 阶段不实现遗物系统）

## 4. 局内系统——时间轴与卡槽

### 4.1 耐力预算

- 每天的可用卡槽总时长 = 耐力属性值（单位：小时）
- 初始耐力因职业而异（约 12~15h）
- 升级耐力 = 每天能探索更多 = 收益更高，但占用了升级其他属性的灵魂

### 4.2 卡槽排列

玩家在规划阶段将卡牌按顺序放入时间轴：

```
[卡牌A: 3h] [卡牌B: 5h] [卡牌C: 2h] [篝火: 2h] [返回: 0h]
|←———————— 白天 ———————→|←———— 夜晚 ————→|
```

- 卡槽数量不固定，取决于放入多少张卡
- 按累计时长推进，到达总耐力 50% 时切换为夜晚
- 最后一张卡之后的剩余耐力 = 休息时间（少量 HP 恢复）

### 4.3 白天 / 夜晚

| 维度 | 白天 | 夜晚 |
|------|------|------|
| 怪物强度 | 基础 | +30%~+50% |
| 灵魂掉落 | 基础 | +40%~+60% |
| 信息清晰度 | 高（可见怪物类型和数量） | 低（部分信息隐藏） |
| 特殊怪 | 普通变种 | 暗影变种（独有掉落） |

- 分界点 = 当天总耐力 × 50%
- 时间操控卡可以改变日夜区间

### 4.4 时间操控卡

| 卡牌 | 时长 | 效果 |
|------|------|------|
| 篝火 | 2h | 回复 HP + 刷新治疗瓶 + 可升级属性，每天限 1 次 |
| 返回卡 | 0h | 立即结束当天，进入结算 |
| 日蚀 | 1h | 接下来 3h 白天区间强制变为夜晚效果 |
| 黎明 | 1h | 接下来 3h 夜晚区间强制变为白天效果 |
| 时间加速 | 0h | 下一张卡时长减半，效果不变 |

## 5. 卡牌系统

### 5.1 三大卡组分类

| 大类 | 包含 | 使用方式 | UI 色系 |
|------|------|---------|--------|
| 路程卡 | 房间、地形、时间操控、返回 | 放入时间轴卡槽 | 蓝 / 青色系 |
| 镶嵌卡 | NPC、事件 | 镶嵌到房间卡内部插槽 | 金 / 琥珀色系 |
| 角色卡 | 装备、战灰、消耗品 | 装备到勇者或背包中 | 红 / 暗色系 |

### 5.2 品质等级

| 品质 | 颜色 | 说明 |
|------|------|------|
| 普通 | 白色 | 基础卡牌，容易获得 |
| 稀有 | 蓝色 | 较强效果，需要一定探索 |
| 史诗 | 紫色 | 强力效果，稀有掉落 / 精英奖励 |
| 传说 | 金色 | 极强效果，极稀有（MVP 阶段不出现） |

房间卡品质决定镶嵌槽的品质上限：
- 白色房间：只能镶嵌白色 NPC / 事件
- 蓝色房间：可镶嵌蓝色及以下
- 紫色房间：可镶嵌紫色及以下
- 金色房间：无限制

### 5.3 卡牌分层——渐进解锁

| 游戏阶段 | 开放层级 |
|---------|---------|
| Day 1~2 | 房间卡（无镶嵌） |
| Day 3~5 | 房间卡 + 镶嵌（NPC ×1 + 事件 ×1） |
| Day 6+ | 地形卡 + 房间卡 + 镶嵌（NPC ×1~2 + 事件 ×1~2） |

解锁时机可以与 Mini-Boss 击败挂钩。

### 5.4 永久 / 消耗分类

| 卡牌类型 | 永久版 | 消耗版 |
|---------|--------|--------|
| 房间卡 | 有（探索度 3 级递进） | 有（一次满探索度体验） |
| NPC 卡 | 有（冷却 1 天） | 有（效果更强，用完消失） |
| 事件卡 | 无 | 只有消耗 |
| 地形卡 | 有 | 有 |
| 装备卡 | 有 | - |
| 消耗品卡 | - | 有 |
| 战灰卡 | 有 | - |
| 时间操控卡 | 有 | 有 |

### 5.5 房间探索度

同一张永久房间卡多次使用的变化机制：

| 探索度 | 内容 |
|--------|------|
| 第 1 次 | 基础怪物 + 基础掉落 |
| 第 2 次 | 解锁隐藏区域（更深层、更强怪物、更好掉落） |
| 第 3 次 | 触发特殊事件（一次性奖励） |
| 第 4 次+ | 效果衰减（灵魂掉落 -30%），鼓励轮换 |

消耗房间卡直接以满探索度状态运行，一次体验完整内容。

### 5.6 卡牌 Combo 系统

| Combo 类型 | 条件 | 效果示例 |
|-----------|------|---------|
| 相邻加成 | 两张特定卡相邻放置 | 不死聚落 + 废弃墓地 = 不死系掉落 ×1.5 |
| 地形连锁 | 连续 3 个相同地形 | 触发地形共鸣（全局属性加成） |
| NPC 组合 | 特定 NPC + 特定房间 | 铁匠 + 矿洞 = 强化费用 -30% |
| 日夜跨越 | 卡牌横跨日夜分界线 | 特殊双重效果 |

## 6. 跨天预布局

### 6.1 核心规则

- 执行当天时，玩家可以编辑**未来任意天**的卡槽
- 只能放入**当前已拥有**的卡牌
- 当天开始执行后**锁定不可更改**

### 6.2 承诺成本——灵魂取回费

| 条件 | 取回费 |
|------|--------|
| Day 1~2 的卡槽 | 免费（新手保护期） |
| Day 3+ 白色卡 | 10 灵魂 |
| Day 3+ 蓝色卡 | 30 灵魂 |
| Day 3+ 紫色卡 | 80 灵魂 |
| Day 3+ 金色卡 | 200 灵魂 |

- 放入免费，取回收费
- 鼓励"想好了再放"，惩罚反复犹豫

### 6.3 UI 交互

屏幕下方可展开/收起日程规划表，显示所有天的卡槽状态：
- 当天：锁定状态（灰色），正在执行
- 未来天：可编辑（高亮边框），显示已放置的卡牌
- Mini-Boss / Boss 天：特殊标记，不可编辑

## 7. 角色成长系统

### 7.1 灵魂——统一货币

| 获取 | 途径 |
|------|------|
| 战斗 | 击杀怪物掉落 |
| 事件 | 骰子成功 / 探索奖励 |
| NPC | 任务奖励 |

| 消费 | 途径 |
|------|------|
| 升级 | 篝火处消耗灵魂提升属性 |
| 购买 | NPC 商店购买卡牌 / 消耗品 |
| 强化 | 铁匠处强化武器 |
| 预布局 | 取回已放置卡牌的费用 |

### 7.2 六维属性

| 属性 | 战斗效果 | 系统效果 | 软上限 |
|------|---------|---------|--------|
| 生命力 | 最大 HP +15/级，自然恢复 +2%/级 | - | 25 |
| 耐力 | - | 每天卡槽时长 +1h/级 | 22 |
| 力量 | 物理伤害 +3%/级 | 背包 +1 格/2 级 | 30 |
| 敏捷 | 闪避率 +1.5%/级，暴击率 +1%/级 | - | 30 |
| 智力 | 魔法伤害 +3%/级 | 骰子修正 +1/5 级 | 30 |
| 信仰 | 治疗瓶效果 +3%/级 | NPC 好感 +2%/级 | 30 |

**升级成本公式**（参考）：

```
升级费 = 基础费 × (当前等级 / 10)^1.5 + 当前等级 × 20

示例（基础费 = 100）：
10级 → 120灵魂
15级 → 190灵魂
20级 → 330灵魂
25级 → 560灵魂
30级 → 900灵魂
```

超过软上限后，每级收益减半但费用继续递增（强烈不建议但不禁止）。

### 7.3 武器系统

**补正等级与系数**：

| 等级 | 系数 |
|------|------|
| S | 1.2 |
| A | 1.0 |
| B | 0.75 |
| C | 0.5 |
| D | 0.25 |
| E | 0.1 |

**伤害计算**（简化版）：

```
最终伤害 = 武器基础攻击 × (1 + 补正系数 × 对应属性 / 50) × 强化加成
强化加成 = 1 + 强化等级 × 0.08
```

**强化系统**：

| 强化段 | 需要材料 | 需要灵魂 |
|--------|---------|---------|
| +1 ~ +3 | 普通石 ×1 | 100 / 200 / 350 |
| +4 ~ +6 | 大块石 ×1 | 500 / 700 / 1000 |
| +7 ~ +10 | 楔形石 ×1 | 1500 / 2000 / 2800 / 4000 |

需要铁匠 NPC 在场才能执行强化。

### 7.4 治疗瓶

| 属性 | 值 |
|------|---|
| 初始瓶数 | 3 |
| 最大瓶数 | 7 |
| 基础回复 | 40% 最大 HP |
| 信仰加成 | +3% / 信仰级 |
| 获取永久 +1 | 特定事件 / 特定房间探索度-3 |
| 刷新方式 | 每天重置 + 篝火卡刷新 |

## 8. 战斗系统

### 8.1 自动战斗

战斗完全自动进行，玩家不直接操控攻击。

**战斗流程**：

```
进入房间卡区间
  → 生成怪物（根据房间 + 白天/夜晚）
  → 勇者自动攻击（普攻 + 特技循环）
  → 怪物自动攻击
  → 击杀怪物 → 掉落物 → 玩家选择保留/分解
  → 触发 NPC（按时间点出现）
  → 触发事件（骰子判定）
  → 房间结束 → 进入下一张卡
```

### 8.2 伤害公式

```
物理伤害 = 攻击力 × (1 - 防御力 / (防御力 + 100))
闪避判定 = rand() < 闪避率 → 伤害为 0
暴击判定 = rand() < 暴击率 → 伤害 × 1.5
```

### 8.3 怪物设计原则

| 类型 | 特征 | 示例 |
|------|------|------|
| 普通怪 | 低血低伤，成群 | 骷髅兵、野狼、史莱姆 |
| 精英怪 | 高血有技能 | 骷髅骑士、岩石巨人 |
| 夜晚变种 | 普通怪的强化版 | 暗影骷髅、黑暗野兽 |
| 精英（结算） | 单独 Boss 级战斗 | 黑骑士、巨型不死龙 |

### 8.4 Boss 战（特殊战斗）

Boss 战是唯一玩家有更多操控权的阶段：

**阶段间策略选择**：

| 选项 | 效果 |
|------|------|
| 全力输出 | 攻击力 +30%，防御 -20% |
| 防御回复 | 防御 +30%，缓慢回复 HP |
| 切换武器 | 更换另一把武器（如果有） |

**Boss 大招应对**：

| 选项 | 效果 |
|------|------|
| 闪避 | 敏捷判定：成功→无伤，失败→全额伤害 |
| 格挡 | 固定减伤 50%，无判定 |
| 使用消耗品 | 视消耗品效果（火焰壶可打断某些大招） |
| 硬扛 | 不操作，吃全额伤害 |

## 9. 死亡与惩罚

### 9.1 魂系一命容错

| 事件 | 效果 |
|------|------|
| 第一次死亡（HP→0） | 当天强制结束，未花费灵魂掉落在死亡卡槽 |
| 回收灵魂 | 下一天安排同一房间卡→进入时自动回收灵魂 |
| 二次死亡（未回收） | 掉落灵魂永久消失 + **Game Over** |
| 二次死亡（已回收） | 重新进入"第一次死亡"状态 |

### 9.2 死亡的间接惩罚

- 浪费了一天的时间（Boss 逼近度仍在增长）
- 当天剩余卡槽的收益全部损失
- 需要用一个卡槽位放同一房间卡来回收灵魂（浪费编排资源）
- 心理压力：二次死亡 = Game Over

## 10. 战役路线与 Boss

### 10.1 路线结构

| 路线 | 天数 | Mini-Boss | 最终 Boss | 难度 |
|------|------|-----------|----------|------|
| 堕落骑士之路 | 4 天 | 无 | 堕落骑士 | 入门 |
| 深渊猎犬之路 | 6 天 | Day 3: 深渊看门犬 | 深渊领主 | 标准 |
| 黑夜君临之路 | 8 天 | Day 3: 看门犬 / Day 6: 黑暗使徒 | 黑夜君主 | 挑战 |

路线在局开始前选定，选择界面展示：
- 天数、Mini-Boss 节点
- 最终 Boss 名称 + 弱点属性提示
- 推荐 Build 方向

### 10.2 Mini-Boss 设计原则

- 2 阶段战斗（比最终 Boss 简短）
- 奖励是**质变性**的（解锁新卡池 / 解锁镶嵌系统 / 核心装备）
- 失败遵循一命容错规则

### 10.3 Boss 逼近度

| 事件 | 逼近度变化 |
|------|-----------|
| 每过一天 | +10 ~ +15（根据路线） |
| 跳过精英战 | +5 |
| 使用强力贪心卡（紫色+） | +2 |
| 击败精英 | -3 |

- **检查时机**：仅在每天结算阶段
- **逼近度 ≥ 100**：强制进入 Boss 战，Boss 获得"逼近强化"（攻防 +15%）
- **逼近度 70~99**：可主动选择挑战（无强化，额外奖励 +20%）

### 10.4 Boss 战结构

```
Boss 出场演出
  → 第一阶段 → 阶段间选择策略
  → 第二阶段 → 阶段间选择策略
  → 第三阶段（最终 Boss 独有）→ 阶段间选择策略
  → 第四阶段（最终 Boss 狂暴期）
  → 击败 → 结算
```

## 13. 随机性与惊喜感

### 13.1 稀有遭遇系统

每个房间卡有 3%~5% 概率触发稀有遭遇（独立于正常事件）：

| 遭遇 | 效果 |
|------|------|
| 神秘商人 | 出售 3 张紫色卡牌，价格极低 |
| 时空裂缝 | 传送到随机高难度区域，击败后获得传说级奖励 |
| 隐藏 Boss | 迷你 Boss 战，掉落独有装备 |
| 篝火幻影 | 免费升级 2 点属性 |

### 13.2 每局诅咒 / 祝福

| 维度 | 规则 |
|------|------|
| 数量 | 每局 1 祝福 + 1 诅咒 |
| 揭示 | 祝福在局开始展示；诅咒在 Day 2 揭示 |
| 移除 | 诅咒可通过特定 NPC / 事件付费移除 |

**祝福示例**：
- 火焰亲和：火属性伤害 +25%
- 商人的恩惠：NPC 商店价格 -30%
- 探索者之眼：所有房间探索度视为 +1

**诅咒示例**：
- 永夜：白天区间缩短为 30%（夜晚占 70%）
- 贪婪之影：灵魂获取 -20%，但掉落卡牌品质 +1 级
- 脆弱：最大 HP -15%，但攻击力 +10%

好的诅咒应改变策略偏好而非纯粹削弱。

## 15. 音频与美术方向

### 15.1 美术风格

- **主基调**：暗黑奇幻（Dark Fantasy）
- **画面风格**：2D 手绘风 / 像素风（根据团队能力选择）
- **调色板**：低饱和度为主，关键元素（灵魂、篝火、Boss）高对比突出
- **参考**：Loop Hero 的简约风 × Dark Souls 的氛围感

### 15.2 音频方向

- **BGM**：低沉弦乐 + 环境音为主，Boss 战切换为激烈战斗曲
- **音效**：攻击、暴击、闪避、升级、骰子、篝火音效
- **参考**：Dark Souls 系列的音乐氛围

### 15.3 文字叙事

- 卡牌描述中嵌入碎片化叙事（魂系传统）
- NPC 对话简短、意味深长
- 不做完整剧情线，通过卡牌描述和环境讲述世界观
