# DS09 - 幽暗编年史：深渊重构 (Chronicles of the Abyss: Reforged)

**设计理念**: 从"计算型扫雷"转向"不可名状的生存恐怖"体验

## 核心改版（相比DS08）

### 1. 威胁等级系统（替代精确数字）
不再显示精确的陷阱数量，改为模糊的威胁感知：

| 等级 | 颜色 | 威胁当量 | 玩家感知 |
|-----|------|---------|---------|
| 🟢 安全 | 绿色 | 0-2 | "这里感觉安全" |
| 🟡 不安 | 黄色 | 3-5 | "有什么东西在附近..." |
| 🔴 危险 | 红色 | 6+ | "极度危险，不要靠近！" |

### 2. 机遇符号系统
每个格子显示第二个维度——可能的回报：

| 符号 | 含义 | 预期 |
|-----|------|------|
| · | 普通 | 可能是空地或陷阱 |
| 👁️ | 异常 | 30%概率是主线/支线房 |
| 📜 | 回声 | 确定有剧情/道具/情报 |

### 3. 遭遇概率机制（替代固定陷阱）
移除预设陷阱，改为动态遭遇：

```
每次移动时进行1d100判定：
- ≤ 当前理智值：安全通过
- > 理智值：遭遇"古老存在"

遭遇后选择：
- 逃跑：理智-10，随机掉落物品
- 对抗：消耗道具，可能受伤
- 凝视：理智-20，获得情报
```

### 4. 搜刮-撤离循环
每层地图随机生成1-2个撤离点（🚪）：

**找到撤离点后选择：**
- ✅ **立即撤离**：带走全部战利品，理智恢复，本层重置
- ⚠️ **继续深入**：无法回头，但收益翻倍

**核心张力**：
- 待越久 → 搜刮越多 → 遭遇概率越高 → 可能全军覆没
- 提前撤 → 安全带走 → 收益有限 → 需要多次探索

### 5. 揭示机制改版
- 踩上去才知道是陷阱还是房间
- 主线/支线房**不预设位置**，探索时随机生成
- 避免玩家背板，增加不可预测性

## 与DS08的关键差异

| 维度 | DS08（经典版） | DS09（重构版） |
|-----|---------------|---------------|
| 核心玩法 | 计算推理扫雷 | 风险评估生存 |
| 信息精度 | 精确数字 | 模糊等级 |
| 恐惧来源 | 踩雷惩罚 | 未知侵蚀 |
| 胜利方式 | 解谜通关 | 搜刮撤离 |
| 玩家能力 | 全知掌控 | 残缺感知 |

## 操作说明

- **左键**：探索格子（揭示威胁等级）
- **右键**：标记异常（消耗标记器，降低遭遇概率）
- **规则**：找到撤离点后才能安全离开本层

## 开发阶段

**Phase 1（MVP）**：威胁等级 + 机遇符号 + 遭遇概率 + 撤离机制
**Phase 2（深度）**：信息碎片 + 诱惑机制 + 幻觉系统
**Phase 3（叙事）**：新剧情文本 + 多结局扩展

## 技术栈

- HTML5 + CSS3 + ES6+
- LocalStorage存档
- 响应式设计
