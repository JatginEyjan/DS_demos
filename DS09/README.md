# DS08 - 幽暗编年史 (Chronicles of the Abyss)

**设计理念**: 融合跑团机制的叙事扫雷游戏

## 核心创新

### 1. 双副本系统
- **副本A：岭下暗影** - 蛇人/隧道/奴隶贸易（初始解锁）
- **副本B：幽暗之门** - 疗养院/罗伊格尔/旧印（需解锁）

### 2. 5层递进地图
| 层级 | 规模 | 探索步数要求 |
|------|------|-------------|
| 1层 | 6×6 | 8步 |
| 2层 | 9×9 | 15步 |
| 3层 | 10×10 | 20步 |
| 4层 | 15×15 | 35步 |
| 5层 | 16×30 | BOSS战 |

### 3. 创新扫雷机制
**数字编码规则：**
- 普通房间 = 周围陷阱数（1-8）
- 主线剧情房 = 陷阱数×10（10,20,30...）
- 支线剧情房 = 陷阱数×(-1)（-1,-2,-3...）

### 4. 标记器系统
- 每次右键标记消耗1个标记器
- 标记正确 → 返还标记器 + 奖励
- 标记错误 → 不返还标记器
- 初始3个，探索中获得

### 5. d100剧情判定
| 操作 | 好走向概率 | 加成因素 |
|------|-----------|---------|
| 左键开启剧情房 | 30% | +理智/10×5% |
| 右键标记剧情房 | 70% | +道具加成 |

### 6. 幻觉模式
- 理智<30时进入
- 数字失真、假剧情房
- d100判定-10%惩罚

## 操作说明

- **左键**：开启格子（普通=踩雷，剧情=坏走向判定）
- **右键**：标记格子（消耗标记器，触发好走向判定）
- **规则**：每层需探索指定步数后才可撤离

## 技术栈

- HTML5 + CSS3 + ES6+
- LocalStorage存档
- 响应式设计
