# DS09 - 幽暗编年史：深渊重构

**核心定位**: 扫雷玩法 + 威胁底色 + 剧情事件 + 撤离机制

## 核心机制

### 1. 基础扫雷玩法
- 点击格子揭示
- **0威胁自动连锁揭示**（像扫雷一样展开安全区）
- 踩到雷 = 触发事件（陷阱或剧情）

### 2. 威胁底色系统（替代扫雷数字）
| 周围雷数 | 显示 |
|---------|------|
| 0 | 默认底色（安全区） |
| 1-2 | 🟡 黄色底色 |
| 3+ | 🔴 红色底色 + 脉动效果 |

### 3. 剧情提示系统
- 周围有剧情房时显示 📜 图标
- 不区分主线/支线（统一图标，后续与叙事强关联）

### 4. 雷的类型（都是雷）
| 类型 | 踩到效果 |
|-----|---------|
| 💀 陷阱 | 扣理智（15-25） |
| 📜 剧情房 | 触发剧情事件，获得收益 |
| 🚪 撤离点 | 可选择撤离或继续 |

### 5. 揭示后的显示
- **普通空地**：显示威胁底色，有剧情则显示📜
- **陷阱**：显示为💀（已触发）
- **剧情房**：显示为📍或🎯（已揭示）
- **撤离点**：显示为🚪

### 6. 撤离机制
找到撤离点后选择：
- **撤离**：安全带走全部收益
- **继续**：收益翻倍，前往下一层（或最终结局）

## 与DS08的差异

| | DS08 | DS09 |
|---|---|---|
| 数字显示 | 精确数字 | 威胁底色（黄/红） |
| 连锁揭示 | 无 | 有（0威胁自动展开） |
| 特殊房间 | 预设位置 | 随机放置（都是雷） |
| 胜利条件 | 探索步数 | 找到撤离点 |

## 体验目标

通过扫雷的"计算+猜测"核心乐趣，配合：
- 威胁底色的视觉压迫感
- 剧情事件的叙事驱动
- 撤离选择的risk-reward抉择

创造"每一步都在赌"的紧张体验。

---

*扫雷的确定性 + 恐怖游戏的不确定性 = 深渊重构*
