# 叙事设计内容文档

> 深渊之下，真相之上，故事在阴影中生长。

## 叙事架构总览

### 主题核心
**"不可名状的恐惧"** —— 不是面对怪物，而是面对无法理解的存在。

### 叙事层次
```
表层：地牢探险的求生故事
中层：调查员逐渐被疯狂侵蚀的心理变化
深层：人类在宇宙中的渺小与无力
```

### 情感弧线
```
好奇 → 不安 → 恐惧 → 执念 → 疯狂/觉醒
```

## 扩展世界观

### 游戏宇宙：《幽暗编年史》

**核心设定：**
我们的世界之下，存在着"深渊"——不是地理概念，而是维度的裂缝。
不同文明、不同时代，都曾有人发现这些裂缝，并试图理解、利用、或封印它们。

**深渊的特性：**
- **不可知性**：无法被完全理解，只能被感知
- **诱惑性**：会向特定的人发出"呼唤"
- **疯狂性**：深入者要么死亡，要么疯狂，极少数"觉醒"

**调查员组织：**
一个松散的秘密组织，成员被称为"调查员"。他们探索深渊遗迹，记录发现，有时也负责封印。
组织内部有等级：见习→正式→资深→守望者

### 副本（章节）体系

**DS09 当前副本：蛇人之遗**
- **主题**：古老的蛇人文明试图通过深渊获得永恒
- **风格**：地下神庙、古老符文、爬行生物
- **核心恐惧**：身体异化（蛇的意象）

**未来可能的副本（缺口/设想）：**

| 副本名 | 主题 | 风格 | 核心恐惧 |
|-------|------|------|---------|
| 深海之渊 | 海底文明 | 水压、深海生物、窒息 | 被吞噬 |
| 虚空灯塔 | 宇宙观测站 | 星空、孤独、时间错乱 | 虚无 |
| 镜像都市 | 平行世界 | 城市、熟悉的陌生、自我 | 身份丧失 |
| 血肉花园 | 生物实验 | 有机体、共生、变异 | 失去人形 |

**跨副本联系：**
- 共同的"深渊"本质
- 调查员的传承（前辈的日记、遗物在不同副本中出现）
- 真相的拼图（每个副本揭示深渊的一个面向）

## 当前副本：蛇人之遗

### 副本背景

**地点：** 东欧某处被遗忘的地下遗迹
**历史：** 数千年前，蛇人文明在此繁荣。他们崇拜深渊中的"伊格"，试图通过仪式获得超越生死的力量。
**现状：** 文明已消亡，但深渊的呼唤仍在。遗迹成为了"有生命"的迷宫，会根据入侵者改变。

### 副本核心主题

**"蜕皮"的隐喻：**
- 蛇人相信通过"蜕皮"可以获得新生
- 但真正的"蜕皮"是抛弃人性，拥抱深渊
- 玩家的旅程也是一次"蜕皮"——要么成为深渊的一部分，要么带着创伤逃离

**关键意象：**
- **蛇**：变形、诱惑、永恒
- **门**：选择、界限、通往未知的入口
- **镜子**：自我、真实与幻觉的边界
- **低语**：深渊的呼唤，理智的挑战

## 主线剧情概括

### 主线结构：五层递进

```
【第一层：初入】
目标：建立好奇心，引入世界观
核心剧情：发现蛇人文明的存在，获得关键道具（古老钥匙）
情感：好奇 > 轻微不安

【第二层：深入】
目标：增加压力，展示代价
核心剧情：遇到前辈的遗骸，意识到危险真实存在
情感：不安 > 恐惧萌芽

【第三层：面对】
目标：获得力量，但也更接近疯狂
核心剧情：获得神秘卷轴，理解蛇人的目标，幻觉开始出现
情感：恐惧 > 执念

【第四层：迷失】
目标：时空错乱，理智挑战
核心剧情：时间循环、镜像自我，分不清真实与幻觉
情感：执念 > 疯狂边缘

【第五层：抉择】
目标：最终抉择，主题升华
核心剧情：面对深渊之门，选择开门（觉醒/崩溃）或逃离（幸存者）
情感：疯狂/觉醒
```

### 主线关键节点

| 节点 | 触发条件 | 剧情内容 | 影响 |
|-----|---------|---------|------|
| 获得古老钥匙 | L1剧情房成功 | 发现蛇人遗迹，获得通往第5层的钥匙 | 解锁真结局可能 |
| 遇见前辈 | L2剧情房 | 发现之前调查员的结局 | 暗示玩家可能的命运 |
| 获得卷轴 | L3剧情房 | 理解蛇人的真相 | 精神力+2，理智-10 |
| 时间错乱 | L4剧情房 | 遇见过去的/未来的自己 | 状态：时间混乱 |
| 最终抉择 | L5深渊祭坛 | 使用钥匙开门或逃离 | 决定结局 |

## 支线剧情分类

### 类型A：环境叙事（发现类）
**特点：** 不需要判定，通过探索发现
**目的：** 丰富世界观，补充背景

**示例：**
- **蛇人壁画**：描述蛇人文明的日常和仪式
- **骸骨堆**：之前探索者的遗骸，暗示危险
- **符文墙**：不可读但可感知的古老文字

**奖励：** 情报、少量金币、无风险

### 类型B：道德抉择（选择类）
**特点：** 需要做出选择，有后果
**目的：** 让玩家参与叙事，影响状态

**示例：**
- **低语回廊**：是否聆听低语？（情报 vs 理智）
- **镜像房间**：是否打碎镜子？（安全 vs 情报）
- **时间回廊**：是否与过去的自己交流？

**后果：** 不同选择获得不同奖励或惩罚

### 类型C：危险探索（判定类）
**特点：** 需要d100判定，高风险高回报
**目的：** 创造紧张感和奖励感

**示例：**
- **地窖入口**：探索隧道可能获得道具或踩陷阱
- **蛇人祭司**：取走卷轴可能成功或被诅咒
- **深渊祭坛**：开门的最终判定

**奖励：** 关键道具、大量资源
**惩罚：** 理智损失、负面状态

### 类型D：幻觉专属（理智类）
**特点：** 理智<30时触发，虚实难辨
**目的：** 强化疯狂感，创造独特体验

**示例：**
- **不存在的门**：看到通往"家"的门
- **过去的你**：看到自己在做还没做的事
- **对话**：与不存在的人对话

**后果：** 通常是惩罚，但可能获得隐藏信息

### 支线与主线的关系

```
主线（必须）：L1 → L2 → L3 → L4 → L5（开门/逃离）
   ↓      ↓      ↓      ↓      ↓
支线（可选）：环境发现 + 道德抉择 + 危险探索 + 幻觉事件
   ↓      ↓      ↓      ↓      ↓
结果：情报、道具、状态影响最终结局
```

## 道具故事

### 设计理念
每个道具都有故事，不仅是功能，也是叙事载体。

### 功能道具

#### 🧪 理智药水
**外观：** 紫色液体，玻璃瓶，散发着草药和某种更古老的气息

**故事：**
```
这是调查员组织的标准配给。
据说配方来自一位"回来"的资深调查员——
他在疯狂边缘被拉回来，然后花了十年时间调配这种药水。

瓶身上贴着褪色的标签：
"记住：这只是暂时的清醒。深渊不会忘记你。"
```

**使用叙事：**
```
你拔开瓶塞，紫色的液体散发着草药和...某种更古老的气息。
喝下的瞬间，你感到混乱的思绪被梳理，疯狂低语暂时退去了。

但你注意到，瓶底有一行小字：
"第47批。使用者：████（名字被划掉）。状态：失效。"

理智+20
```

---

#### 🔍 探测器
**外观：** 老式的金属仪器，指针不断抖动，表面刻着你不认识的刻度

**故事：**
```
这是从"深海之渊"副本回收的设备。
据说它能探测"异常"，但代价是使用者的...某些东西。

之前的拥有者在日记中写道：
"它越来越准了。但我也越来越不像自己了。"
```

**使用叙事：**
```
你打开探测器，指针疯狂旋转后指向某个方向。
当你看向那个方向时，指针突然静止。

你感觉到某种东西从探测器流向那个格子——
是信息？还是你的...注意力？

随机揭示1格
```

---

#### 💠 灵能水晶
**外观：** 半透明的蓝色晶体，内部似乎有液体流动

**故事：**
```
蛇人文明的遗物。他们相信水晶是"凝固的深渊"，
可以用来储存和转移精神力量。

这块水晶中的液体...是蓝色的。
但蛇人壁画中的水晶，液体是红色的。

也许这块还没有被"污染"。
也许。
```

**使用叙事：**
```
你握住水晶，感受到一股清凉的能量流入体内。
水晶中的液体减少了一些。

你突然意识到——这些液体是什么？
是能量？还是...别的什么东西的提取物？

精神力+2
```

---

#### 🏮 煤油灯
**外观：** 老式煤油灯，但火焰是冷的，不会烧伤手

**故事：**
```
这盏灯来自第一个探索蛇人遗迹的调查员。
他在第3层失去了理智，但在彻底疯狂前把这盏灯留在了安全的地方。

灯上的刻字：
"光只是暂时的庇护。黑暗才是永恒的真相。
但有时候，暂时的庇护就是一切。"
```

**使用叙事（退出幻觉）：**
```
你点燃煤油灯，冷色的火焰照亮周围。
疯狂开始退去，墙壁不再蠕动，低语声消失。

但当你看向灯的火焰时，
你看到了一张脸——不是你自己的——在火焰中一闪而过。

退出幻觉模式
理智+10
```

## 叙事风格指南

### 克苏鲁黄金法则

1. **不可名状**：不说名字，只说特征
   - ❌ "怪物有8只眼睛"
   - ✅ "你感觉到很多视线，但数不清楚"

2. **感官优先**：用感官描述，不用概念
   - ❌ "这里很危险"
   - ✅ "空气变得黏稠，呼吸需要用力"

3. **个人化**：用"你"，让玩家代入
   - 避免旁观者的客观描述

4. **留白**：不说完，让玩家想象
   - 暗示 > 明示

5. **反差**：成功也有不安，失败也有信息
   - 没有纯粹的好或坏

### 道具叙事原则

1. **来源故事**：这个道具从哪来？之前的主人怎么了？
2. **使用代价**：功能越强，暗示的代价越大
3. **未知元素**：保留一些说不清的细节
4. **情感连接**：让玩家对道具产生情感（喜爱、恐惧、好奇）
