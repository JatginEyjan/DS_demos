# 叙事设计内容文档

> 深渊之下，真相之上，故事在阴影中生长。

---

## 叙事架构总览

### 主题核心
**"不可名状的恐惧"** —— 不是面对怪物，而是面对无法理解的存在。

### 叙事层次
```
表层：地牢探险的求生故事
中层：调查员逐渐被疯狂侵蚀的心理变化
深层：人类在宇宙中的渺小与无力
```

### 情感弧线
```
好奇 → 不安 → 恐惧 → 执念 → 疯狂/觉醒
```

---

## 世界观设定

### 背景：蛇人文明的遗迹

**设定：**
数千年前，一个名为"伊格"的蛇人文明在此地繁荣。他们崇拜深渊中的存在，建造了这座地下神庙。文明的突然消失留下了这座诅咒之地，任何深入的人都会听到"呼唤"。

**关键元素：**
- **蛇人文字**：墙壁上的符文，不可读但可感知含义
- **深渊祭坛**：第5层的核心，通往"门"的所在
- **调查员前辈**：之前的探索者留下的痕迹（日记、尸体、遗物）

### 主角：无名的调查员

**身份：** 
你是一名接到委托的调查员，任务探索这片被诅咒的土地。

**动机：**
- 表层：金钱、名誉
- 中层：对未知的好奇
- 深层：某种"呼唤"在吸引你

**变化：**
- 第1层：自信、好奇
- 第3层：疲惫、不安
- 第5层：疯狂或觉醒

## 剧情房设计

### 设计原则

每个剧情房包含：
1. **场景设定**：环境描写，营造氛围
2. **选择/判定**：d100骰子决定结果
3. **分支结果**：成功/失败两种叙事
4. **情感反差**：成功也有不安，失败也有信息

### 叙事风格

**克苏鲁黄金法则：**
- ❌ 不说："你看到怪物有8只眼睛"
- ✅ 说："你感觉到很多视线，但你数不清楚，因为当你看向它们时，它们在移动"

**感官优先：**
- 触觉：冰凉、黏腻、刺痛
- 嗅觉：霉味、血腥、古老
- 听觉：低语、滴答、心跳
- 视觉：模糊、闪烁、余光中的移动

## 第1层剧情房

### L1_01: 地窖入口

**场景：** 老宅地窖

**行动前：**
```
推开老宅沉重的木门，穿过积灰的走廊来到地窖，
潮湿的霉味混杂着泥土气息扑面而来。
墙角的砖墙被破开一个可供人通行的大洞，
里面吹来一阵潮湿且带有淡淡腥味的微风...

你可以：
1. 握紧手电筒，小心探索（判定）
2. 先观察洞口周围（理性+5，跳过本房）
```

**判定：d100 ≤ 35 成功**

**成功：**
```
preText: "你握紧手电筒，小心翼翼地向隧道深处探索..."
resultText: "隧道尽头是一个不大的石室，中央摆放着一盏老式的煤油灯。\n\n灯旁有一张泛黄的纸条：'当你看到这盏灯时，记住——光只是暂时的庇护，黑暗才是永恒的真相。'\n\n获得：煤油灯 🏮"
奖励：理智+10，获得煤油灯
```

**失败：**
```
preText: "你刚踏入隧道几步，黑暗中突然有什么冰凉的东西擦过你的肩膀。\n你猛地转身...却什么也没看到。但那刺骨的寒意真实存在。\n当你再次转身时，你发现来时的路看起来变长了..."
resultText: "你跌跌撞撞地退出隧道，但那种被注视的感觉一直跟着你。\n理智-15，获得状态：不安"
惩罚：理智-15
```

你可以：

2. 研究骸骨指向的方向（情报+1）

**判定：d100 ≤ 40 成功**

**成功：**
```
preText: "你强忍着恶心，小心地在骸骨堆中搜索..."

## 第2层剧情房

### L2_01: 低语回廊

**场景：** 狭长的回廊

**行动前：**
```
这条回廊似乎没有尽头。
更可怕的是，你开始听到低语声。
不是来自某个方向，而是来自所有方向，
包括你脑海深处。

"回来..."
"加入我们..."
"你听到了吗..."

你可以：
1. 捂住耳朵前进（判定）
2. 尝试聆听低语的内容（高风险，可能获得情报）
3. 使用煤油灯（如果有）强制安静
```

**判定：d100 ≤ 35 成功**

**成功：**
```
preText: "你捂住耳朵，强迫自己不去听那些声音，一步一步向前..."
resultText: "低语声逐渐变弱，你看到了回廊的尽头——一扇石门。\n门上刻着：'只有拒绝诱惑者，才能通过。'\n\n理智-5（低语的残留影响）\n获得：地图碎片（显示本层部分布局）"
奖励：地图碎片，理智-5
```

**失败：**
```
preText: "低语声越来越响，你开始分不清哪些是你的想法，哪些是它们的声音..."
resultText: "当你回过神来时，你发现自己站在回廊的起点。\n但墙上多了一个手印——是你的，但你的手上没有灰尘。\n那个"你"是什么时候留下的？\n理智-20，强制传送至随机位置"

```

## 第3层剧情房

### L3_01: 镜像房间

**场景：** 布满镜子的房间

**行动前：**
```
房间四周布满了古老的镜子。
但镜子中的景象...不对劲。
有的镜子显示的房间比实际更大，
有的镜子中有人在移动但你看不到，
有的镜子中...你看到了自己，但那个"你"在微笑，而你并没有。

你可以：
1. 尝试打碎镜子（判定，高风险）
2. 寻找不反光的镜子（判定，中等风险）
3. 闭上眼睛摸索离开（安全但可能迷路）
```

**判定：d100 ≤ 35 成功**

**成功：**
```
preText: "你仔细观察每一面镜子，寻找那个'不反光'的..."
resultText: "你找到了。\n那面镜子显示的不是反射，而是另一个房间——一个你还没有到达的房间。\n通过这个'窗口'，你看到了第五层的部分布局。\n\n获得情报：第五层布局预览\n理智-10（窥视未知的代价）"
奖励：第五层情报，理智-10
```

**失败：**
```
preText: "当你看向镜子时，镜子中的'你'开口说话了——"
resultText: "'你来得太慢了，'它说，'我已经等你很久了。'\n然后，它从镜子中伸出了手，触碰了你的脸。\n那一瞬间，你看到了无数可能性：你死在这里，你逃出去了，你成为了它...\n当你回过神，你独自站在房间中央，所有镜子都碎了。\n理智-30，获得状态：预见（某些时候会提前看到陷阱）"

## 第5层剧情房

### L5_01: 深渊祭坛（BOSS房）

**场景：** 最终祭坛

**行动前：**
```
你终于到达了最深处。
这里是一个巨大的地下空洞，中央是一个祭坛。
祭坛上不是神像，而是一扇门。
一扇通往何处的门？

门上有一个锁孔，形状和你手中的古老钥匙一模一样。
但你突然意识到——你可以打开它，但你不确定是否应该。

你可以选择：
1. 使用钥匙打开门（进入真结局判定）
2. 调查祭坛周围（判定，可能获得情报或道具）
3. 寻找其他出路（可能触发普通结局）
```

**使用钥匙判定：d100 ≤ 50 成功（需要古老钥匙）**

**成功（真结局 - 觉醒）：**
```
preText: "你将钥匙插入锁孔，转动..."
resultText: "门开了。\n\n门后不是另一个房间，是星空。\n无尽的、疯狂的、美丽的星空。\n你看到了真相——人类在宇宙中的位置，你的存在意义，一切的开始和结束。\n\n这种知识超出了人类的承受极限。\n\n你疯了。\n但你在疯狂中获得了平静。\n\n【真结局：觉醒】\n理智归0，但你理解了。"
特殊结局：游戏通关，解锁隐藏内容
```

**失败（真结局 - 崩溃）：**
```
preText: "你将钥匙插入锁孔，转动..."
resultText: "门开了一条缝。\n\n从缝隙中涌出的不是光，是知识——以痛苦的形式。\n你看到了你不该看到的东西，理解了你不该理解的事物。\n\n你尖叫着关门，但已经太晚了。\n\n你疯了。\n彻底的、无法恢复的疯狂。\n\n【真结局：崩溃】\n理智归0，你被深渊吞噬。"
特殊结局：游戏通关，但角色永久疯狂
```

**无钥匙选择调查：**
```
preText: "你小心地调查祭坛周围..."
resultText: "你在祭坛底部发现了一个暗格。\n里面有足够的补给让你安全撤离。\n\n获得：大量金币和道具\n\n当你转身离开时，你听到门后传来一声叹息——\n不是失望，而是...期待？\n\n【普通结局：幸存者】\n你逃出去了，带着财富和秘密。\n但你知道，有一天你会回来。\n门在等你。"
普通结局：正常通关
```

## 幻觉专属剧情（理智<30）

### 设计原则
- 幻觉时可能看到"不存在"的剧情房
- 这些剧情房的文本更扭曲、更个人化
- 玩家无法分辨这是真实还是幻觉

### 示例：不存在的门

**触发条件：** 理智<30，随机触发

**场景：** 
```
你看到了一扇门。
它不在地图上，不在网格中，
但你知道它通向你的家。
你的床，你的窗户，你的安全。

你可以选择：
1. 推开门（？？？）
2. 忽视它，继续前进（理智-10，痛苦但清醒）
```

**选择推开门：**
```
preText: "你推开了门..."
resultText: "门后是你的卧室。\n阳光从窗帘透进来，你的猫在床上睡觉。\n一切如此真实。\n\n你躺在床上，闭上眼睛。\n\n【游戏结束】\n\n当你在现实中醒来时，你发现自己站在地牢深处，
面前是一堵墙，你的额头满是血迹。\n\n理智-50，位置随机重置"
惩罚：理智-50，传送至随机位置
```
