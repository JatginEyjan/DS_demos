# DS09 - 幽暗编年史：深渊重构

> 从精确计算到模糊恐惧，从掌控一切到在未知中前行。
> 在不可名状的恐惧中，用有限的理智换取真相，在贪婪与安全之间做出抉择。

---

## 这是一个怎样的游戏

**一句话：** 一款融合扫雷推理与克苏鲁恐怖的地牢探索游戏。

**核心体验：** 在信息不完全的黑暗中，凭借碎片线索推理危险，每一步都要权衡"继续探索的贪婪"与"理智崩溃的风险"，最终在深渊吞噬你之前找到生路。

**从DS08到DS09的演进：**
- DS08：精确数字让玩家"掌控一切"，恐惧不足 ❌
- DS09：风险底色提供"大致感知"，保留不确定性 ✅
- 核心转变：**从"计算安全"到"感知危险"，从"解谜"到"生存"**

## 系统框架

```
┌─────────────────────────────────────────────────────────┐
│                    【核心循环】                           │
│                                                         │
│  探索网格 → 推理威胁 → 做出抉择 → 承受后果 → 评估状态    │
│      ↑________________________________________↓        │
└─────────────────────────────────────────────────────────┘

┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│   信息层     │    │   决策层     │    │   后果层     │
│              │    │              │    │              │
│ • 威胁底色   │───→│ • 是否揭示   │───→│ • 安全通过   │
│ • 👁️注视提示 │    │ • 精神扫描   │    │ • 踩到陷阱   │
│ • 理智状态   │    │ • 立即撤离   │    │ • 发现剧情   │
└──────────────┘    └──────────────┘    └──────────────┘
```

### 核心循环详解

**1. 探索网格（信息收集）**
- 点击未揭示的格子
- 根据周围危险程度，格子显示不同底色
- 🟢安全(0威胁) → 🟡不安(1-2威胁) → 🔴危险(3+威胁)
- 周围有剧情房时显示👁️，暗示"这里值得冒险"

**2. 推理威胁（策略形成）**
- 根据底色和👁️提示，推断周围陷阱和剧情房的位置
- 决定是否使用有限的精神力💠进行安全扫描
- 评估当前理智值能否承受可能的损失

**3. 做出抉择（风险权衡）**
- **贪婪选项**：继续探索，追求更多剧情和收益
- **安全选项**：寻找撤离点，带走已获资源
- **激进选项**：用精神力扫描高风险区域，换取安全

**4. 承受后果（反馈体验）**
- 踩到陷阱：理智下降，距离疯狂更近一步
- 发现剧情：获得叙事体验和随机奖励
- 连锁揭示：成功推理后的爽快感

**5. 评估状态（循环继续）**
- 理智还够吗？幻觉开始了吗？
- 达到撤离标准了吗？要贪一波还是撤？

## 为什么好玩

### 乐趣1：不完全信息的推理快感
**不是扫雷的精确计算，而是风险估算。**

你知道这里有危险（🔴），但不知道具体在哪。你有线索（👁️暗示附近有剧情），但不确定位置。每一次点击都是一次赌博，而你的推理能力让你赌得更聪明。

**爽点：** 通过👁️+🔴的组合，成功预判陷阱位置，然后用💠精神扫描安全规避。

### 乐趣2：资源管理的紧张感
**精神力💠和理智🧠都是有限资源，用一点少一点。**

- 精神力可以让你安全扫描，但用光了就只能硬闯
- 理智让你保持清醒，但踩到陷阱或看到恐怖场景会下降
- 幻觉模式：理智<30时，你看到的底色可能是假的...

**紧张时刻：** 只剩1点精神力，面前是🔴危险区域，你敢不敢直接点？

### 乐趣3：贪婪vs安全的永恒抉择
**撤离机制创造的"最后一搏"时刻。**

达标后，撤离按钮一直在那，你可以选择：
- **现在走**：安全但收益有限
- **再探几格**：可能大赚，也可能理智崩溃前功尽弃

**戏剧性：** 理智只剩10点，还差2步就能发现那个👁️剧情房，你按不按撤离？

### 乐趣4：克苏鲁式的无力感
**信息永远不完全，危险永远存在。**

- 你不可能知道所有陷阱位置
- 幻觉会让你误判
- 即使成功撤离，你知道下次可能没那么幸运

**氛围：** 不是jump scare，而是持续的不安——那种"我是不是漏掉了什么"的感觉。

## 叙事期望

### 叙事定位
**"调查员笔记"式的碎片化叙事**

玩家扮演一名调查员，深入被诅咒的地下遗迹。叙事不是线性故事，而是通过：
- **剧情房触发的事件**：每个房间都是一个独立的小场景
- **行动前/行动后的文本**：选择前后的叙事反差
- **道具描述**：带出血肉模糊的线索
- **环境暗示**：网格中的👁️、幻觉时的扭曲

### 叙事体验目标
| 情感 | 叙事手段 | 当前状态 |
|-----|---------|---------|
| **好奇** | 👁️提示的若隐若现 | ✅ 已实现 |
| **不安** | 底色变化暗示危险临近 | ✅ 已实现 |
| **恐惧** | 幻觉时的界面扭曲+文字 | ⚠️ 需强化 |
| **沉浸** | 剧情房的完整叙事（preText+resultText） | ✅ 已实现 |
| **无力感** | 即使成功也暗示更大的恐怖 | ❌ **缺口** |

### 叙事缺口
1. **深层叙事缺失**：目前剧情房是独立事件，缺少跨层的连贯线索
2. **结局冲击力不足**：撤离成功缺少"代价感"，玩家应该感到"逃出来了，但已经不同了"
3. **幻觉叙事薄弱**：幻觉目前只有视觉扭曲，应该有专属的剧情文本和选择

## 三大系统支柱

### 支柱1：威胁感知系统（信息层）
**功能：** 将精确的危险转化为模糊的风险等级

| 元素 | 作用 | 体验 | 缺口 |
|-----|------|------|------|
| 威胁底色🟢🟡🔴 | 提示周围危险程度 | 紧张但可推理 | 幻觉模式下是否应有专属底色？ |
| 👁️注视提示 | 暗示附近有剧情房 | 好奇与恐惧并存 | 是否有不同等级的注视？ |
| 幻觉误导 | 理智<30时底色可能错误 | 失控感 | 幻觉是否应该有专属剧情？ |

### 支柱2：理智与精神系统（资源层）
**功能：** 限制玩家行为，创造"心有余而力不足"的体验

| 资源 | 作用 | 消耗场景 | 恢复方式 | 缺口 |
|-----|------|---------|---------|------|
| 理智🧠 | 健康值，0=游戏结束 | 踩陷阱、剧情失败 | 撤离恢复、道具 | 理智低到阈值时是否有特殊事件？ |
| 精神力💠 | 安全扫描次数 | 右键扫描 | 灵能水晶道具 | 精神力是否有其他用途？ |

**缺口思考：**
- 理智和精神力目前只有"消耗"，缺少"转化"或"牺牲换收益"的抉择
- 例如：主动降低理智换取更强的精神扫描？

### 支柱3：撤离抉择系统（目标层）
**功能：** 提供明确的"退出点"，但每次退出都有机会成本

- **硬性目标：** 探索X步后解锁撤离
- **软性诱惑：** 超额探索有额外收益
- **风险递增：** 待得越久，理智消耗越大

**缺口：**
- 收益目前是线性的，是否可以非线性（指数增长）来增强贪婪的诱惑？
- 撤离后是否可以保留部分"记忆"影响下一层？

## 体验缺口（待解决）

### 🔴 高优先级
| 缺口 | 影响 | 可能方案 |
|-----|------|---------|
| **难度曲线太平缓** | 无法进入心流，容错率过高导致随机体验无有效反馈 | 动态难度、陷阱密度递增、关键节点强制挑战 |
| **幻觉模式冲击力不足** | 削弱恐怖核心体验 | 幻觉专属剧情、更严重的界面扭曲 |
| **剧情房吸引力不够** | 削弱探索动力 | 不同等级的👁️提示、更丰厚的叙事奖励 |
| **撤离抉择张力不足** | 削弱核心抉择体验 | 非线性收益、撤离后的代价叙事 |
| **叙事连贯性缺失** | 削弱沉浸感 | 跨层线索、结局的代价描述 |

### 🟡 中优先级
| 缺口 | 影响 | 可能方案 |
|-----|------|---------|
| **0威胁连锁的爽感** | 削弱推理正向反馈 | 更强动画+音效、连锁计数奖励 |
| **精神力的消耗感** | 资源系统单一 | 精神力换理智、精神力保留到下一层？ |
| **结局的情感冲击** | 削弱整体记忆点 | 多结局、结局后的"调查员笔记" |

### 🟢 低优先级/探索性
| 缺口 | 影响 | 可能方案 |
|-----|------|---------|
| **多层之间的关联** | 缺少长线策略 | 跨层道具、层间叙事线索 |
| **难度曲线** | 第1层和第5层体验差异 | 动态难度、可选的挑战模式 |

## 游戏体验时间线

```
【开局】谨慎探索，学习威胁模式
   ↓
【中期】开始推理，尝试高效路径，面临第一次撤离抉择
   ↓
【后期】理智下降，幻觉出现，每个决定都关乎生死
   ↓
【终局】在疯狂边缘找到撤离点，带走战利品或葬身深渊
```

**单场游戏时长：** 15-30分钟（一层）
**完整体验：** 5层递进，每层地图和叙事都不同

## 成功标准

**如果DS09成功了，玩家会说：**
- "这个底色系统让我紧张，但又忍不住想探索"
- "我知道可能有陷阱，但那个👁️让我好奇"
- "最后那几步，我真的在考虑要不要撤"
- "幻觉模式让我 panic 了"
- **（期望新增）** "虽然逃出来了，但感觉已经被改变了"

**数据指标（如有）：**
- 平均探索率（是否敢于超额探索？）
- 精神扫描使用时机（是否用于关键决策？）
- 剧情房触发率（👁️提示是否有效？）
- **（期望新增）** 结局后的情感反馈（是否感到"代价"？）

## 设计边界

### ✅ 我们要做
- 扫雷的核心推理循环（揭示→连锁→推理）
- 克苏鲁氛围的营造（信息模糊、幻觉、理智机制）
- 风险抉择（撤离机制带来的张力）
- 叙事与机制的结合（剧情房触发事件）

### ❌ 我们不做
- 复杂的战斗系统（保持扫雷的简洁）
- 多人联机（专注单机体验）
- 永久死亡roguelike（保留层数继承，降低挫败感）
- 精确数值计算（用底色模糊化，强化不确定感）

3. **假设C**：如果剧情房有不同的"诱惑等级"（👁️→✨→🔮），会增加探索动力吗？

4. **假设D**：如果结局后有"调查员笔记"（描述精神创伤），会增强代价感吗？
