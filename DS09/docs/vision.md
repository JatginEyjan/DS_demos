# DS09 - 幽暗编年史：深渊重构

> 在不可名状的恐惧中，用有限的理智换取真相，在贪婪与安全之间做出抉择。

---

## 这是一个怎样的游戏

**一句话：** 一款融合扫雷推理与克苏鲁恐怖的地牢探索游戏。

**核心乐趣：** 在信息不完全的黑暗中，凭借碎片线索推理危险，每一步都要权衡"继续探索的贪婪"与"理智崩溃的风险"，最终在深渊吞噬你之前找到生路。

---

## 系统框架

```
┌─────────────────────────────────────────────────────────┐
│                    【核心循环】                           │
│                                                         │
│  探索网格 → 推理威胁 → 做出抉择 → 承受后果 → 评估状态    │
│      ↑________________________________________↓        │
└─────────────────────────────────────────────────────────┘

┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│   信息层     │    │   决策层     │    │   后果层     │
│              │    │              │    │              │
│ • 威胁底色   │───→│ • 是否揭示   │───→│ • 安全通过   │
│ • 👁️注视提示 │    │ • 精神扫描   │    │ • 踩到陷阱   │
│ • 理智状态   │    │ • 立即撤离   │    │ • 发现剧情   │
└──────────────┘    └──────────────┘    └──────────────┘
```

### 核心循环详解

**1. 探索网格（信息收集）**
- 点击未揭示的格子
- 根据周围危险程度，格子显示不同底色
- 🟢安全(0威胁) → 🟡不安(1-2威胁) → 🔴危险(3+威胁)
- 周围有剧情房时显示👁️，暗示"这里值得冒险"

**2. 推理威胁（策略形成）**
- 根据底色和👁️提示，推断周围陷阱和剧情房的位置
- 决定是否使用有限的精神力💠进行安全扫描
- 评估当前理智值能否承受可能的损失

**3. 做出抉择（风险权衡）**
- **贪婪选项**：继续探索，追求更多剧情和收益
- **安全选项**：寻找撤离点，带走已获资源
- **激进选项**：用精神力扫描高风险区域，换取安全

**4. 承受后果（反馈体验）**
- 踩到陷阱：理智下降，距离疯狂更近一步
- 发现剧情：获得叙事体验和随机奖励
- 连锁揭示：成功推理后的爽快感

**5. 评估状态（循环继续）**
- 理智还够吗？幻觉开始了吗？
- 达到撤离标准了吗？要贪一波还是撤？

## 为什么好玩

### 乐趣1：不完全信息的推理快感
**不是扫雷的精确计算，而是风险估算。**

你知道这里有危险（🔴），但不知道具体在哪。你有线索（👁️暗示附近有剧情），但不确定位置。每一次点击都是一次赌博，而你的推理能力让你赌得更聪明。

**爽点：** 通过👁️+🔴的组合，成功预判陷阱位置，然后用💠精神扫描安全规避。

### 乐趣2：资源管理的紧张感
**精神力💠和理智🧠都是有限资源，用一点少一点。**

- 精神力可以让你安全扫描，但用光了就只能硬闯
- 理智让你保持清醒，但踩到陷阱或看到恐怖场景会下降
- 幻觉模式：理智<30时，你看到的底色可能是假的...

**紧张时刻：** 只剩1点精神力，面前是🔴危险区域，你敢不敢直接点？

### 乐趣3：贪婪vs安全的永恒抉择
**撤离机制创造的"最后一搏"时刻。**

达标后，撤离按钮一直在那，你可以选择：
- **现在走**：安全但收益有限
- **再探几格**：可能大赚，也可能理智崩溃前功尽弃

**戏剧性：** 理智只剩10点，还差2步就能发现那个👁️剧情房，你按不按撤离？

### 乐趣4：克苏鲁式的无力感
**信息永远不完全，危险永远存在。**

- 你不可能知道所有陷阱位置
- 幻觉会让你误判
- 即使成功撤离，你知道下次可能没那么幸运

**氛围：** 不是jump scare，而是持续的不安——那种"我是不是漏掉了什么"的感觉。

## 三大系统支柱

### 支柱1：威胁感知系统（信息层）
**功能：** 将精确的危险转化为模糊的风险等级

| 元素 | 作用 | 体验 |
|-----|------|------|
| 威胁底色🟢🟡🔴 | 提示周围危险程度 | 紧张但可推理 |
| 👁️注视提示 | 暗示附近有剧情房 | 好奇与恐惧并存 |
| 幻觉误导 | 理智<30时底色可能错误 | 失控感 |

### 支柱2：理智与精神系统（资源层）
**功能：** 限制玩家行为，创造"心有余而力不足"的体验

| 资源 | 作用 | 消耗场景 | 恢复方式 |
|-----|------|---------|---------|
| 理智🧠 | 健康值，0=游戏结束 | 踩陷阱、剧情失败 | 撤离恢复、道具 |
| 精神力💠 | 安全扫描次数 | 右键扫描 | 灵能水晶道具 |

### 支柱3：撤离抉择系统（目标层）
**功能：** 提供明确的"退出点"，但每次退出都有机会成本

- **硬性目标：** 探索X步后解锁撤离
- **软性诱惑：** 超额探索有额外收益
- **风险递增：** 待得越久，理智消耗越大

## 游戏体验时间线

```
【开局】谨慎探索，学习威胁模式
   ↓
【中期】开始推理，尝试高效路径，面临第一次撤离抉择
   ↓
【后期】理智下降，幻觉出现，每个决定都关乎生死
   ↓
【终局】在疯狂边缘找到撤离点，带走战利品或葬身深渊
```

**单场游戏时长：** 15-30分钟（一层）
**完整体验：** 5层递进，每层地图和叙事都不同

---

## 这不是什么

- ❌ **不是精确解谜**：没有唯一正确答案，风险估算各有策略
- ❌ **不是动作游戏**：没有反应速度要求，全是决策
- ❌ **不是纯roguelike**：有层数继承，降低挫败感
- ❌ **不是恐怖 jump scare**：恐惧来自氛围和不确定性，而非惊吓

---

## 一句话总结

**"每次点击都是赌博，每次扫描都是权衡，每次撤离都是不甘。在深渊中，最可怕的从不是死亡，而是明知可能死，却依然选择贪婪的那个自己。"**
