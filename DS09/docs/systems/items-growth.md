# 道具与成长系统

> 深渊中的物品，既是工具，也是诅咒的载体。

---

## 设计意图

### 解决什么问题？
- **纯机制游戏缺少收集乐趣**：玩家需要"获得感"
- **缺少长期成长感**：每次探索都是独立的，缺少积累
- **策略深度不足**：需要更多决策维度（带什么道具？怎么用？）

### 创造什么体验？
- **收集乐趣**：发现稀有道具的惊喜
- **策略搭配**：不同道具组合产生不同策略
- **风险抉择**：道具占用背包空间，带什么放弃什么？

### 支撑什么情感？
- **准备感**：进入地牢前的装备选择（像真正的调查员）
- **依赖性**：关键时刻靠道具救命
- **失去恐惧**：死亡时道具丢失的危机感

## 道具系统框架

### 当前状态

```
【进入地牢前】商店购买 → 道具进入背包
         ↓
【地牢中】使用道具 → 效果触发 → 消耗/保留
         ↓
【撤离时】道具结算 → 剧情道具保留/功能道具变现？
         ↓
【死亡时】地牢内获得道具丢失，商店购买道具保留？
```

**当前缺口：**
- 道具结算规则不清晰
- 剧情道具缺少实际功能
- 没有道具组合/升级机制

## 商店系统（当前实现）

### 刷新机制
```javascript
// 进入地牢前刷新
items = [
  3个随机功能道具,  // 必有
  20%概率1个剧情道具  // 稀有
]
```

**问题：**
- 刷新太随机，玩家可能刷不到想要的道具
- 是否应该根据玩家进度调整商店内容？

### 价格体系

| 道具 | 价格 | 性价比 |
|-----|------|--------|
| 灵能水晶 | 30 | 高（精神力珍贵） |
| 理智药水 | 50 | 中 |
| 煤油灯 | 80 | 低（仅在幻觉时有价值） |
| 探测器 | 100 | 中（随机性） |

**考虑调整：**
- 是否应该动态定价？（急需时更贵）
- 是否应该引入"讨价还价"机制？

## 成长系统（缺口）

### 当前状态：无成长
- 每层独立，道具不保留到下一层
- 没有永久解锁的能力或道具
- 缺少长期动力

### 期望的成长方向

#### 🔴 短期成长（层间继承）
**问题：** 目前道具每层重置，缺少积累感

**可能方案：**
1. **剧情道具保留**：发现的剧情道具可以带到下一层
2. **精神力上限提升**：通关一层后，下一层初始精神力+1
3. **情报继承**：发现的情报（如地图布局）部分保留？

#### 🟡 中期成长（跨局积累）
**问题：** 每次游戏都是独立的

**可能方案：**
1. **调查员等级**：多次游戏后解锁新能力
2. **永久解锁道具**：某些道具购买后永久可用（但有限制）
3. **图鉴收集**：收集所有剧情道具解锁隐藏结局

#### 🟢 长期成长（Meta进度）
**探索性：**
1. **多周目剧情**：第二次游玩看到不同的剧情文本
2. **真相拼图**：多次游戏拼凑完整的故事
3. **自定义调查员**：选择不同的背景和能力

## 道具功能扩展（缺口）

### 剧情道具的功能化

**当前：** 剧情道具只有描述和卖金币

**期望：**

| 道具 | 期望功能 |
|-----|---------|
| 古老钥匙 🗝️ | 在特定层解锁隐藏房间（额外剧情+奖励） |
| 神秘卷轴 📜 | 触发特殊剧情分支（不同结局） |
| 护身符 ✨ | 被动：陷阱理智伤害-10% |
| 奴隶地图 🗺️ | 显示本层所有撤离点位置 |

### 新道具构想

#### 功能性
- **时间沙漏**：暂停理智流失5回合
- **记忆水晶**：死亡时保留1个道具
- **鲜血契约**：用理智换精神力（比例2:1）

#### 叙事性
- **破碎的镜子**：携带时幻觉效果增强，但可能看到隐藏信息
- **无名之书**：每次阅读随机获得信息或损失理智
- **熟悉的信物**：来自上一局死亡的调查员，携带时某些剧情改变

## 道具与叙事结合

### 道具获取叙事
**当前缺失：** 道具只是"获得"，没有场景

**期望：**
```
【获得古老钥匙】

你在骸骨堆中发现了一个锈蚀的铁盒。
打开时，里面躺着一枚铜钥匙，上面刻着你不认识的符文。
当你触碰它时，钥匙微微发热，仿佛还有生命...

获得：古老钥匙 🗝️
（这枚钥匙可以打开某些被封锁的门）
```

### 道具使用叙事
**当前缺失：** 使用道具只有效果，没有描述

**期望：**
```
【使用理智药水】

你拔开瓶塞，紫色的液体散发着草药和...某种更古老的气息。
喝下的瞬间，你感到混乱的思绪被梳理，疯狂低语暂时退去了。

理智+20
```

## 关键数值

### 当前数值

| 参数 | 当前值 | 问题 |
|-----|--------|------|
| 背包容量 | 无限制 | 应该有限制，增加选择压力 |
| 初始金币 | 待确认 | 应该够买1-2个道具 |
| 道具价格 | 30-300 | 剧情道具太贵，难以收集 |
| 结算比例 | 未明确 | 撤离时道具怎么处理？ |

### 考虑的数值调整

1. **背包限制**：最多携带5个道具，迫使玩家选择
2. **道具重量**：剧情道具重1，功能道具重0.5，有限负重
3. **动态价格**：根据玩家金币数量调整价格（防止囤钱）

## 与其他系统的关联

### × 商店系统
- 商店是道具的主要获取途径
- 商店刷新机制影响道具策略

### × 剧情系统
- 剧情房奖励道具
- 道具可以触发隐藏剧情

### × 撤离系统
- 撤离时道具结算（哪些带走？哪些变现？）
- 死亡时道具丢失惩罚

### × 理智/精神系统
- 某些道具影响理智或精神力
- 某些道具需要消耗理智使用

## 待办清单

### 🔴 高优先级
1. **剧情道具功能化**：给每个剧情道具实际效果
2. **道具结算规则明确**：撤离/死亡时道具怎么处理
3. **道具获取叙事**：获得和使用时有叙事描述

### 🟡 中优先级
4. **背包限制**：引入容量或重量限制
5. **层间继承**：剧情道具是否可以保留到下一层
6. **商店优化**：动态价格、根据进度调整商品

### 🟢 低优先级
7. **道具组合**：某些道具组合使用有特殊效果
8. **Meta成长**：多周目解锁、调查员等级
9. **新道具设计**：时间沙漏、记忆水晶等

## 设计原则总结

1. **道具是策略维度**：不仅是工具，更是决策（带什么？什么时候用？）
2. **稀缺性创造选择**：背包有限，必须取舍
3. **叙事与功能结合**：道具既是游戏机制，也是故事载体
4. **成长感**：短期（层间继承）和长期（Meta进度）都要有
