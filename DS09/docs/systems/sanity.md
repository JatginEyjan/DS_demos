# 理智系统

> 你的理智是唯一的光，在深渊中，它既是武器，也是燃料。

---

## 设计意图

### 解决什么问题？
- **纯扫雷只有输赢**：没有渐进的压力积累
- **恐怖游戏只有惊吓**：缺少持续的心理压迫
- **需要一种"缓慢燃烧"的紧张感**：玩家知道自己在走向疯狂

### 创造什么体验？
- **倒计时感**：理智在流逝，必须尽快撤离
- **风险计算**：这个格子值得我损失多少理智？
- **失控边缘**：理智低时的幻觉，信息不再可信

### 支撑什么情感？
- **脆弱感**：你不是无敌的调查员，你的精神有限
- **紧迫感**：时间不多了
- **疯狂**：当理智崩溃，你看到了不存在的东西

## 核心规则

### 理智值范围
```
0 - 100

100 = 完全清醒
30 = 幻觉临界点
0 = 精神崩溃，游戏结束
```

### 理智消耗场景

| 场景 | 理智变化 | 说明 |
|-----|---------|------|
| 踩到陷阱 | -15~25 | 随机伤害，增加不确定性 |
| 剧情失败 | -10~-20 | 根据剧情严重程度 |
| 剧情成功 | +5~+10 | 好结果恢复部分理智 |
| 撤离成功 | 恢复到100 | 逃离后精神恢复 |
| 幻觉持续 | -1/回合（考虑中） | 待在幻觉中越久越疯狂？ |

**实现状态：** ✅ 已实现基础（踩陷阱/剧情失败扣除，道具恢复）
**当前缺口（未实现）：**
- 持续理智流失（深层地牢每步流失）❌
- 理智阈值动态调整（深层更早触发幻觉）❌
- 理智崩溃特殊事件 ❌

### 理智恢复方式

| 方式 | 恢复量 | 限制 |
|-----|--------|------|
| 理智药水 | +20 | 消耗品，数量有限 |
| 剧情成功 | +5~+10 | 随机，不可控 |
| 撤离 | 100 | 必须放弃当前探索 |
| 标记陷阱成功 | +5 | 小量奖励 |

**缺口：**
- 恢复方式有限，且大多不可控
- 是否可以有"短暂休息"机制？（消耗时间换取理智恢复）

## 幻觉模式

### 触发条件
```javascript
if (sanity < 30) {
  进入幻觉模式();
}
```

### 当前效果
- **视觉扭曲**：30%概率显示错误的威胁底色
- **界面变化**：边框颜色改变、轻微抖动动画

**当前缺失：** 幻觉只有视觉效果，没有叙事支撑

**期望：**
```

你开始看到一些...不存在的东西。
墙壁上的符文在你余光中蠕动，但当你直视时，它们静止不动。

【理智 < 20】

低语声。不是来自某个方向，而是来自所有方向，
包括你脑海深处。

【理智 < 10】

你看到了一个门。它不在地图上，不在网格中，
```

## 理智与游戏节奏

### 理智作为"软倒计时"

```
游戏初期（100-70理智）：
  自信探索，敢于冒险
  幻觉未触发，信息可信

游戏中期（70-30理智）：
  开始谨慎，计算风险
  接近临界点，压力增加

游戏后期（<30理智）：
  幻觉干扰，信息不可靠
  必须依靠推理和运气
  撤离 vs 贪婪的最终抉择

崩溃（0理智）：
  游戏结束
  失去所有副本内收益
```

**设计意图：**
- 理智创造"软倒计时"，不是时间限制，而是错误容忍度限制
- 玩家可以主动选择"用理智换信息"（精神扫描）
- 高理智 = 掌控感，低理智 = 失控感

- **深层更难恢复**：深层理智药水效果减半？
- **幻觉阈值变化**：深层更早触发幻觉？

## 与其他系统的关联

### × 威胁感知系统
- 理智<30触发幻觉，威胁底色可能错误
- 理智越低，信息越不可靠

### × 精神扫描系统
- 精神扫描是"用另一种资源换信息"，绕过理智限制
- 但幻觉严重时，扫描结果也可能错误

### × 撤离系统
- 理智是"能否继续探索"的硬性限制
- 撤离是理智耗尽前的"安全出口"

### × 叙事系统
- 理智值影响剧情文本（低理智时文本扭曲）
- 理智崩溃应该有特殊结局叙事

## 设计原则总结

1. **理智是另一种生命值**：但比生命值更"软"，更有策略性
2. **信息即生存**：理智低时信息不可靠，创造"明知有险但必须赌"的张力
3. **缓慢燃烧**：不是瞬间死亡，而是逐渐失控的过程
4. **疯狂的诱惑**：是否可以有"拥抱疯狂换取力量"的选择？
