## 1. 游戏概述

### 1.1 核心体验
玩家扮演一名**蒸汽机甲技师**，在失控的齿轮工厂中生存30分钟。通过自动武器击杀敌人获取经验，升级选择新武器和强化，在越来越强的机械敌潮中走位生存。

**一句话描述**: *蒸汽朋克风格的吸血鬼幸存者，加入冲刺闪避机制，30分钟生存挑战。*

### 1.2 核心乐趣
- **Build构筑**: 6种武器组合，不同流派（近战环绕、远程轰炸、召唤流）
- **操作感**: 冲刺无敌帧带来的精准躲避快感
- **成长反馈**: 从弱小到割草的指数级变强
- **目标明确**: 30分钟倒计时，每局有清晰终点

## 2. 核心玩法

### 2.1 基础操作

| 操作 | PC | 移动端 |
|------|-----|--------|
| 移动 | WASD | 左侧虚拟摇杆 |
| 冲刺 | 空格键 | 右侧冲刺按钮 |
| 攻击 | 自动 | 自动 |

**冲刺机制**:
- 效果: 向当前移动方向冲刺150像素，0.3秒无敌帧
- CD: 基础4秒（受升级影响）
- 视觉: 黄铜色蒸汽残影，角色闪烁
- 战术: 穿怪、躲技能、紧急捡经验

### 2.2 核心循环

```
移动走位 → 自动击杀 → 掉落经验齿轮 → 升级3选1 → 构筑Build → 应对更强敌人 → 存活30分钟
```

### 2.3 胜利条件
- **普通胜利**: 存活满30分钟
- **完美胜利**: 30分钟内不死亡（触发隐藏Boss）

## 3. 角色系统

### 3.1 基础属性

| 属性 | 基础值 | 成长方式 |
|------|--------|----------|
| 生命值 | 100 | 升级+20/级，道具加成 |
| 移动速度 | 150px/s | 升级+10%，道具加成 |
| 冲刺CD | 4秒 | 升级-0.5秒/级，最多2秒 |
| 经验拾取范围 | 80px | 升级+20%，道具加成 |
| 武器槽位 | 6个 | 固定 |

### 3.2 可解锁角色（局外）

| 角色 | 解锁条件 | 特性 |
|------|----------|------|
| **工程师** | 初始 | 平衡型，无特殊 |
| **锅炉工** | 存活10分钟 | +30%血量，-10%移速 |
| **电工** | 击杀1000个敌人 | 特斯拉武器初始自带，+20%电击伤害 |
| **飞艇手** | 冲刺100次 | +50%冲刺距离，CD 3秒 |
| **炼金术士** | 通关一次 | 药水效果+50%，开局带1瓶血药 |

## 4. 武器系统

### 4.1 武器类型

**可同时装备最多6种，自动攻击，无需瞄准**

| 武器 | 类型 | 基础效果 | 进化效果 |
|------|------|----------|----------|
| **旋转齿轮** | 近战/环绕 | 2个齿轮环绕身体，每秒20伤害 | 锯齿风暴：4个齿轮，范围变大，伤害+50% |
| **蒸汽喷射** | 近战/后方 | 向后喷射蒸汽，击退+减速身后敌人 | 高压蒸汽：范围变宽，附加燃烧伤害 |
| **压力榴弹** | 远程/范围 | 每2秒投掷1颗榴弹，爆炸范围100px | 连环爆破：爆炸后分裂3个小炸弹 |
| **特斯拉无人机** | 远程/追踪 | 1架无人机自动电击最近敌人 | 蜂群：3架无人机，可同时攻击 |
| **地刺管道** | 远程/随机 | 每3秒从地面随机刺出铜管 | 管道网络：刺出频率翻倍，范围+30% |
| **齿轮飞镖** | 远程/穿透 | 每1秒向前方发射穿透飞镖 | 风暴：发射3发扇形飞镖 |

### 4.2 升级稀有度

| 稀有度 | 出现概率 | 效果加成 |
|--------|----------|----------|
| 普通 | 60% | 标准效果 |
| 稀有 | 30% | +30%伤害/效果 |
| 史诗 | 9% | +60%伤害，附加特效 |
| 传说 | 1% | +100%伤害，改变攻击模式 |

## 5. 敌人设计

| 敌人 | 血量 | 速度 | 伤害 | 特性 |

| 15分钟 | **失控机甲** | 发射追踪导弹，可用冲刺无敌帧躲 |
| 20分钟 | **齿轮女王** | 召唤小怪+全屏齿轮雨，考验走位 |
| 25分钟 | **锅炉之心** | 血量极高，多种攻击模式混合 |
| 30分钟 | **完美通关隐藏Boss** | 可选挑战，击败解锁特殊奖励 |

## 6. 升级系统

### 6.1 升级选择（局内）

**每升一级，游戏暂停，3选1**

#### 选项类型
1. **新武器** - 获得一个随机武器（未满6个时优先出现）
2. **武器进化** - 随机一个已有武器进化（已满级则换其他）
3. **属性强化** - 生命值/移速/冲刺CD/拾取范围
4. **特殊道具** - 血瓶/临时无敌/全屏清怪

#### 属性强化表

| 强化项 | 每级效果 | 上限 |
|--------|----------|------|
| 最大生命 | +20 | 无上限 |
| 生命恢复 | +1/秒 | 10/秒 |
| 移动速度 | +10% | +100% |
| 冲刺CD | -0.3秒 | 最低1.5秒 |
| 拾取范围 | +20% | +200% |
| 经验获取 | +15% | +150% |

### 6.2 局外成长（工坊系统）

**每局获得「黄铜齿轮」，用于永久解锁**

| 类别 | 内容 | 消耗 |
|------|------|------|
| **角色** | 解锁新角色 | 500-2000齿轮 |
| **武器图鉴** | 新武器加入池子 | 300齿轮/武器 |
| **开局加成** | +10%初始移速、+20初始生命等 | 200-1000齿轮 |
| **便利功能** | 开局跳过前5分钟、自动拾取经验 | 800-1500齿轮 |
| **皮肤** | 角色外观变化（纯视觉） | 500齿轮 |

## 7. 难度曲线（30分钟）

### 7.1 时间分段

| 阶段 | 时间 | 敌人强度 | 经验倍率 | 特色 |
|------|------|----------|----------|------|
| 教学期 | 0-5分钟 | 只有齿轮鼠 | 1.0x | 熟悉操作 |
| 成长期 | 5-10分钟 | 加入傀儡、无人机 | 1.2x | Build成型 |
| 压力期 | 10-20分钟 | 加入蜘蛛、坦克 | 1.5x | 需要走位 |
| 生存期 | 20-30分钟 | 全敌种+精英怪 | 2.0x | 极限挑战 |

### 7.2 敌人生成表

```
0-5分钟:   齿轮鼠，每波5-10只，波次间隔10秒
5-10分钟:  齿轮鼠+傀儡，每波8-15只，间隔8秒
10-15分钟: 加入无人机+蜘蛛，每波12-20只，间隔6秒
15-20分钟: 加入自爆锅炉，每波15-25只，间隔5秒
20-25分钟: 加入蒸汽坦克，每波20-30只，间隔4秒
25-30分钟: 全敌种混合，精英怪出现率+50%，间隔3秒
```

## 8. 美术风格

### 8.1 视觉基调
- **色彩**: 黄铜金、铜绿、煤灰黑、蒸汽白
- **像素规格**: 16x16或32x32角色，16色限制营造复古感
- **特效**: 粒子蒸汽、电火花、黄铜光晕

### 8.2 UI设计
- **HUD**: 顶部经验条、左侧倒计时、右侧武器栏
- **升级界面**: 蒸汽朋克齿轮卡牌风格
- **字体**: 像素风格，支持中文

### 8.3 场景
- **背景**: 巨大的齿轮工厂，缓慢转动的机械装置
- **地面**: 金属网格+管道纹理
- **氛围**: 飘浮的蒸汽粒子，偶尔的电火花

## 9. 技术规格

### 9.1 技术栈
- **引擎**: Phaser 3 (v3.60+)
