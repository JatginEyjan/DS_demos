# DS07 - 你就是地牢 (You Are The Dungeon)

**核心理念**: 终极融合 - 你的死亡定义了地牢本身

## 核心机制

### 死亡即地牢

每次死亡：
1. 尸体留在死亡地点
2. 下局探索时，尸体成为特殊格子
3. 可以选择**祭祀**（祝福）或**亵渎**（力量）

### 道德选择

遇到自己的尸体时：
- ✨ **祭祀** - 恢复20理智，尊重死者
- 😈 **亵渎** - 损失15理智，但立即揭示3x3区域

你的选择会累积，影响最终结局

### 最终Boss战

死亡5次后解锁：
- Boss是"你"的累积
- 生命 = 死亡次数 × 10
- 每次揭示格子对Boss造成伤害
- 全部揭示 = 击败自己

## 设计意图

**元游戏体验**
- 你既是探索者，也是地牢的建造者
- 死亡不再是失败，是内容的生成
- 最终面对自己的所有选择

**多结局**
- 祭祀多：净化结局
- 亵渎多：深渊结局  
- 击败Boss：和解结局
