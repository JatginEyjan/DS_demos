## 三、伤害计算公式

### 完整伤害公式
```
实际伤害 = (基础伤害 + 力量) × 易伤倍数 × 虚弱倍数
```

### 攻击流程（实操顺序）
```
1. 打出攻击牌 → 显示基础伤害
2. +力量加成
3. 检查目标是否易伤 → ×1.5
4. 检查攻击者是否虚弱 → ×0.75
5. 计算最终伤害 → 减去目标当前格挡值
6. 格挡抵消后剩余伤害扣HP
```

### 格挡机制
- 格挡在**你的下回合开始时**清除
- 格挡不会跨回合保留（除非有Barricade等特殊效果）
- 格挡优先于HP扣减

## 九、DBG核心设计原则

### 1. 卡牌不是越多越好
- 精选15-20张 > 臃肿30+张
- Strike/Defend需要被移除才有质变
- 1费牌占比高保证每回合都能做事

### 2. 能量管理是核心
- 3能量/回合是基准
- 0费卡战略价值高：可打出多张其他牌
- 过量能量溢出是浪费

### 3. 牌库循环节奏
- 25张典型牌库：抽5张后剩20张
- 第一轮不会抽空
- 疲劳压力在后期出现

### 4. 敌人意图驱动决策
- 玩家知道敌人下回合干什么
- 预判+规划 = 策略感

### 5. 奖励要有选择
- 3选1体现玩家意志
- 没有"正确答案"的抉择才有意义

### 6. 失败要能复盘
- 能说出"哪里出了问题"
- 复盘后想重开

## 十、DS19迭代对照表

### 需要继承的部分
| STS设计 | DS19如何继承 |
|---------|-------------|
| 意图系统 | 必须保留 |
| 3能量/回合 | 保持 |
| 5卡/回合抽牌 | 保持（双角色各抽） |
| 卡牌3类型 | 保持：攻击/技能/能力 |
| 稀有度框架 | 保持 |
| 战斗奖励3选1 | 保持（升级时触发） |
| 初始套牌框架 | 保持精简版 |

### 可以变化的部分
| STS设计 | DS19变化点 |
|---------|-----------|
| 单角色 | 改为双角色切换 |
| 遗物 | 改为装备（每角色1件） |
| 升级靠卡牌 | 改为经验升级+技能选择 |
| 路线规划 | 改为波次推进 |
| 失败条件 | 改为全角色濒死 |

### 绝对不能变的
- **能量消耗出牌** → DBG核心
- **抽牌/弃牌循环** → 构筑感基础
- **意图可预判** → 策略性来源
- **失败可复盘** → Roguelike本质

## 十二、战斗流程时序

```
【玩家回合开始】
  ↓
1. 获得3点能量
2. 每名存活角色抽5张（DS19改为4张）
3. 玩家行动：出牌/切换/使用道具
4. 结束回合
  ↓
【弃掉剩余手牌】
  ↓
【敌方回合】
  ↓
5. 敌人按意图行动（显示下回合意图）
6. 回合结束效果（灼烧等）
  ↓
【循环回到玩家回合】
```
