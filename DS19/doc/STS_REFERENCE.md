## 一、基础数值锚点

### 玩家基础资源
| 资源 | 数值 | 说明 |
|------|------|------|
| 每回合能量 | 3点 | 可消耗、可溢出（不保留到下回合） |
| 每回合抽牌 | 5张 | 弃牌堆洗牌形成抽牌堆 |
| 初始HP | 70-80 | Ironclad 80HP, Silent/Defect 70HP |
| 初始金币 | 99 | Demo默认值 |
| 初始能量 | 3 | 每回合固定回复 |

### 初始套牌结构（Ironclad示例）
```
Strike × 4    — 1费, 6伤害, 攻击
Defend × 4    — 1费, 5格挡, 技能
Bash × 1      — 2费, 8伤害, 赋予2层易伤, 技能
```
**关键**: 初始牌库10张，Strike占4/10 = 40%，所以初期过牌很重要

## 三、状态效果完整列表

### 增益类
| 状态 | 符号 | 效果 | 持续 |
|------|------|------|------|
| 力量 (Strength) | ⚔️+数字 | 每点+1伤害/每次攻击 | 永久直到清除 |
| 敏捷 (Dexterity) | 🛡️+数字 | 每点+1格挡/每张格挡牌 | 回合结束时清除 |
| 虚弱 (Weak) | ⚠️ | 攻击伤害×0.75 | 3回合 |
| 易伤 (Vulnerable) | ❗ | 受伤伤害×1.5 | 3回合 |
| 护甲 (Plating) | 🛡️ | 回合结束额外+3格挡 | 永久 |
| 充能 (Ritual) | ⚡ | 下X回合获得Y力量 | 持续X回合 |

### 减益类
| 状态 | 符号 | 效果 | 持续 |
|------|------|------|------|
| 脆弱 (Frail) | 💀 | 格挡效果-25% | 1回合 |
| 中毒 (Poison) | ☠️ | 每回合开始扣X血，无视格挡 | 持续X回合 |
| 灼烧 (Burn) | 🔥 | 每回合扣X血，可被格挡 | 2回合 |
| 蹒跚 (Vulnerable) | ❗ | 同上易伤 | - |
| 麻痹 (Paralyzed) | ⚡ | 50%概率无法行动 | 1-2回合 |
| 混乱 (Confusion) | ❓ | 攻击随机目标 | 1-3回合 |

### 特殊状态
| 状态 | 效果 |
|------|------|
|  artifacts | 免疫下一次debuff |
|  格挡 (Block) | 抵消等量伤害，下回合开始时清除 |
| 荆棘 (Thorns) | 受到攻击时反弹X伤害 |
| 再生 (Regen) | 每回合恢复X HP |

### 状态叠加规则
- 同名状态通常叠加（力量+1 + 力量+1 = 力量+2）
- 不同状态独立生效
- 护甲类状态通常不叠加，取最大值

## 四、伤害计算公式

### 完整伤害公式
```
最终伤害 = (基础伤害 + 力量) × 易伤倍数 × 虚弱倍数 - 目标格挡
```

### 攻击流程（实操顺序）
```
1. 打出攻击牌 → 显示基础伤害
2. +力量加成 → 中间值
3. 检查目标是否易伤 → ×1.5
4. 检查攻击者是否虚弱 → ×0.75
5. 计算最终伤害 → 减去目标当前格挡值
6. 格挡抵消 → 剩余伤害扣HP
7. 若格挡≥伤害 → 不扣HP,格挡清零
```

### 特殊情况
- **中毒伤害**：无视格挡，直接扣HP
- **灼烧伤害**：可被格挡，每回合结算
- **荆棘反弹**：格挡后仍然触发

### 格挡机制
- 格挡在**你的下回合开始时**清除
- 格挡不会跨回合保留（除非有Barricade等特殊效果）
- 格挡优先于HP扣减

## 五、敌人Intent意图系统

### Intent显示机制
敌人Intent显示在敌人头顶，玩家可提前看到下回合行动：

| Intent图标 | 含义 | 玩家应对 |
|------------|------|----------|
| ⚔️12 | 攻击造成12伤害 | 准备格挡或击杀 |
| 🛡️8 | 获得8格挡 | 可忽略或用破甲 |
| ⬆️+3⚔️ | 获得3层力量后攻击 | 优先击杀 |
| ❓ | 随机意图 | 无法预测 |
| 3×6 | 3次攻击，每次6伤害=18总伤害 | 准备足够格挡 |

### 敌人行动循环示例

**普通敌人循环（典型3-4回合循环）：**
```
回合1: 🛡️12 → 获得格挡
回合2: ⚔️10 → 攻击
回合3: 🛡️8 → 获得格挡
回合4: ⚔️12 → 攻击
(循环)
```

**Gremlin Nob（精英，Act1常见）：**
```
回合1-2: ⬆️+2⚔️ → 获得力量
回合3+: ⚔️18-22 → 高伤害攻击
特点：无格挡，纯攻击压力
```

**Slime Boss（Act1 Boss）：**
```
阶段1（HP>50%）:
  回合1: 🛡️ + 准备（下回合分裂）
  回合2: ⚔️25 + 分裂（变成2个75HP小史莱姆）
阶段2（HP<50%）:
  每个小史莱姆独立循环
```

**Guardian（Act1 Boss）：**
```
回合1: 🛡️30 → 高额护甲
回合2: ⚔️32 → 破甲攻击
回合3: ⚔️8×3 → 三连击24伤害
(循环，但护甲会累加)
```

### Intent对玩家的战略价值
1. **预判防御需求**：看到下回合攻击意图，提前准备格挡
2. **预判斩杀线**：计算敌人剩余HP，规划击杀时机
3. **识别强化敌人**：看到⬆️意图，优先击杀防止滚雪球
4. **识别AOE压力**：看到多段攻击意图，准备群体格挡

## 十一、DS19迭代对照表

### 需要继承的部分
| STS设计 | DS19如何继承 |
|---------|-------------|
| Intent系统 | 必须保留，玩家需要预判敌人行动 |
| 3能量/回合 | 保持 |
| 5卡/回合抽牌 | 保持（双角色各抽4张） |
| 卡牌3类型 | 保持：攻击/技能/能力 |
| 稀有度框架 | 保持：Common/Uncommon/Rare |
| 战斗奖励3选1 | 保持（升级时触发） |
| 初始套牌框架 | 保持精简版 |
| 状态效果系统 | 保持（易伤/虚弱/格挡/力量等） |

### 可以变化的部分
| STS设计 | DS19变化点 |
|---------|-----------|
| 单角色 | 改为双角色切换 |
| 遗物 | 改为装备（每角色1件） |
| 升级靠卡牌 | 改为经验升级+技能选择 |
| 路线规划 | 改为波次推进 |
| 失败条件 | 改为全角色濒死 |

### 绝对不能变的
- **能量消耗出牌** → DBG核心
- **抽牌/弃牌循环** → 构筑感基础
- **Intent可预判** → 策略性来源
- **失败可复盘** → Roguelike本质

## 十三、战斗流程时序

```
【你的回合开始】
  ↓
1. 获得3点能量
2. 抽5张牌（弃牌堆洗入抽牌堆）
3. 玩家行动：出牌/使用道具/切换角色
4. 玩家结束回合
  ↓
【弃掉剩余手牌】
  ↓
【敌方回合】
  ↓
5. 触发回合开始效果（再生、充能等）
6. 敌人按Intent行动（显示下回合Intent）
7. 触发回合结束效果（灼烧1层、中毒扣血等）
  ↓
【你的回合开始】
```

### 关键时序点
- **格挡清除**：你的下回合开始时
- **虚弱/易伤递减**：回合结束时-1层
- **灼烧/中毒**：敌方回合开始时结算
- **再生**：你的回合开始时结算
