## P0 - 核心系统（阻塞性）

### Mission 1: 双角色战斗系统 (v0.1.0)
**优先级**: 🔴 P0 - 阻塞
**状态**: [x] 已完成 (2024-04-12)
**预估工时**: 8h
**引用**: [v0.1.0](../doc/version/v0.1.0.md)

#### 需求描述
实现双角色同时在场、独立资源、自由切换的核心战斗循环

#### 验收标准
- [ ] 双角色同时显示在战斗场景（战士+游侠）
- [ ] 角色A的能量消耗不影响角色B（各3点/回合）
- [ ] 角色A的抽牌来自A的牌库，角色B同理（各4张/回合）
- [ ] 点击切换按钮可切换当前操作角色
- [ ] 切换不消耗能量，切换后手牌区域更新
- [ ] 当前角色有明显高亮标识
- [ ] 回合循环正确：回复能量→抽牌→出牌→弃牌→敌方回合
- [ ] 敌人意图显示伤害数值和攻击目标
- [ ] 敌人全灭→胜利；双角色死亡→失败

#### 技术要点
```typescript
// BattleSystem 核心类
class BattleSystem {
  characters: Character[]  // 2名角色
  currentCharacterIndex: number
  enemies: Enemy[]
  
  switchCharacter(index: number)  // 切换角色
  playCard(card: Card, target: Target)  // 出牌
  endTurn()  // 结束回合，进入敌方回合
}

class Character {
  hp: number
  maxHp: number
  energy: number
  maxEnergy: number
  hand: Card[]
  drawPile: Card[]
  discardPile: Card[]
  
  drawCards(count: number)
  gainEnergy(amount: number)
  takeDamage(amount: number)
}
```

### Mission 2: 卡牌与战斗逻辑 (v0.1.0)
**优先级**: 🔴 P0 - 阻塞
**状态**: [x] 已完成 (2024-04-12)
**预估工时**: 6h
**引用**: [v0.1.0](../doc/version/v0.1.0.md), [卡牌基线](../doc/STS_REFERENCE.md#card-baseline)

#### 需求描述
实现基础卡牌系统、伤害计算、状态效果

#### 验收标准
- [ ] 1费攻击牌：造成6伤害（Strike/射击）
- [ ] 1费防御牌：获得6护甲（Defend/闪避）
- [ ] 卡牌可点击打出，能量不足时灰显
- [ ] 伤害公式：(基础+力量)×易伤×虚弱-格挡
- [ ] 格挡在回合开始时清除
- [ ] 易伤：受到伤害+50%，持续2回合
- [ ] 虚弱：造成伤害-25%，持续2回合
- [ ] 抽牌堆空时，洗弃牌堆继续抽

#### 卡牌配置
```json
{
  "strike": {
    "name": "斩击",
    "cost": 1,
    "type": "attack",
    "effect": { "damage": 6 }
  },
  "defend": {
    "name": "格挡",
    "cost": 1,
    "type": "skill",
    "effect": { "block": 6 }
  }
}
```

- [ ] 升级时弹出三选一界面

- [ ] 选中卡牌加入对应角色牌库
- [ ] 升级时最大生命+4（战士+5），当前生命+8
- [ ] 新卡牌在下一场战斗可被抽到
- [ ] 死亡角色仍获得经验

#### 升级曲线
| 等级 | 累计XP | 生命成长 |
|------|--------|----------|
| Lv1 | 0 | - |
| Lv2 | 2 | +4/+5 |
| Lv3 | 5 | +4/+5 |
| Lv4 | 9 | +4/+5 |
| Lv5 | 14 | +4/+5 |
