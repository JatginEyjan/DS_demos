# DS19 Demo 设计方案
## Dungeon Spirit 19 - 地牢精灵

---

## 一、核心概念定位

### 1.1 游戏类型
Roguelike Deckbuilder + Monster Collection RPG

### 1.2 核心体验
**"收集-培养-构筑"三位一体循环**
- 收集：捕获野生精灵 / 孵化精灵蛋
- 培养：战斗积累经验，升级获得新技能卡
- 构筑：多精灵队伍 + 牌库动态调整

### 1.3 与杀戮尖塔的差异点
| 维度 | 杀戮尖塔 | DS19 |
|------|----------|------|
| 角色 | 单角色，固定职业 | 多精灵队伍，可随时替换 |
| 生命值 | 单一血条 | 每个精灵独立HP，全灭即败 |
| 牌库 | 单一套牌 | 每只精灵贡献独立子牌库，战斗时混合抽牌 |
| 成长 | 遗物+卡牌 | 经验升级+技能学习+装备 |
| 获取 | 卡牌商店/奖励 | 捕获系统+孵化系统 |

## 二、战斗系统框架

### 2.1 基础循环（继承STS）
```
抽牌 → 耗能出牌 → 计算伤害/格挡/效果 → 敌人行动 → 回合结束（格挡清零）
```

### 2.2 多精灵战斗机制

#### 出战机制
- **前排精灵**：承受伤害，可使用所有手牌
- **后排精灵**：每回合只能使用1张牌（辅助/治疗）
- **切换行动**：消耗1点能量，将前后排互换（当回合触发）

#### 牌库构成
- 每位精灵自带独立"技能牌库"
- 战斗时混合所有出战精灵的牌库
- 手牌上限：10张（与STS一致）

#### 能量系统
- 每回合基础能量：3点（可通过装备/能力增加）
- 能量跨精灵共享

### 2.3 状态系统（参考宝可梦属性）

#### 基础属性
- **HP**：生命值，归零即濒死
- **ATK**：影响攻击牌伤害
- **DEF**：影响获得格挡值
- **SPD**：影响抽牌数量/先手判定

#### 异常状态（持续效果）
| 状态 | 效果 | 解除方式 |
|------|------|----------|
| 中毒 | 每回合损失5%最大HP | 药水/特定技能 |
| 麻痹 | 50%概率无法行动 | 3回合后自动解除 |
| 灼烧 | 每回合损失ATK伤害 | 水系技能/道具 |
| 冰冻 | 无法行动 | 火系技能/随机解除 |
| 睡眠 | 无法行动，每回合恢复10%HP | 3回合后或受到伤害 |

## 三、精灵系统

### 3.1 精灵基础结构
```yaml
精灵:
  名称: 火焰幼龙
  属性: 火
  稀有度: 稀有
  基础HP: 60
  基础ATK: 12
  基础DEF: 8
  基础SPD: 10
  
  初始技能牌库:
    - 火焰爪击 x2
    - 蓄力 x2
    - 撞击 x3
    - 防御 x3
  
  升级技能池:
    - 火焰喷射
    - 龙之怒
    - 火焰护盾
    - 烈焰冲锋
    ...
```

### 3.2 经验与升级

#### 经验获取
- 普通战斗：+20 EXP
- 精英战斗：+50 EXP
- Boss战：+100 EXP
- 捕捉成功：+10 EXP（参与战斗的精灵）

#### 升级机制
- 每级需求经验 = 当前等级 × 25
- **升级奖励**：3选1技能牌
  - 选项A：攻击型技能
  - 选项B：防御/辅助型技能
  - 选项C：特殊/稀有技能

#### 等级上限
- Demo版本：精灵最高20级
- 每5级解锁一个"特性槽位"（被动能力）

### 3.3 属性克制系统（简化版）

Demo版本使用5属性循环克制：
```
火 → 草 → 水 → 火
光 ↔ 暗（互克）
```

| 攻击方 ↓ / 防御方 → | 火 | 水 | 草 | 光 | 暗 |
|---------------------|----|----|----|----|----|
| 火 | - | 0.5x | 2x | 1x | 1x |
| 水 | 2x | - | 0.5x | 1x | 1x |
| 草 | 0.5x | 2x | - | 1x | 1x |
| 光 | 1x | 1x | 1x | - | 2x |
| 暗 | 1x | 1x | 1x | 2x | - |

*克制时伤害×2，被克制时伤害×0.5*

## 四、获取系统

### 4.1 精灵捕获系统

#### 捕获道具
| 道具名称 | 效果 | 价格 |
|----------|------|------|
| 普通精灵球 | 基础捕获率 | 50金币 |
| 高级精灵球 | 捕获率+30% | 150金币 |
| 超级精灵球 | 捕获率+60% | 400金币 |
| 大师球 | 100%捕获 | Boss掉落/稀有事件 |

#### 捕获机制
```
捕获成功率 = 基础率(30%) + 削弱加成 + 道具加成

削弱加成:
- HP < 50%: +10%
- HP < 25%: +20%
- 异常状态: +15%
- 睡眠/冰冻: +25%

最高成功率: 95%（保留一点随机性）
```

#### 捕获流程
1. 战斗中选择"捕获"选项
2. 选择要使用的精灵球
3. 播放捕获动画（3次晃动判定）
4. 成功：精灵加入队伍
5. 失败：消耗精灵球，继续战斗

### 4.2 精灵蛋系统

#### 获取途径
- **商店购买**：随机属性蛋（200-500金币）
- **事件奖励**：特定剧情事件
- **战斗掉落**：稀有敌人掉落

#### 孵化机制
- 每个蛋有"孵化值"（如：需要5场战斗）
- 携带蛋进行战斗，每场战斗+1孵化进度
- **篝火选项**："孵化" - 直接完成孵化（消耗篝火回合）

#### 蛋的属性
- 蛋会显示"属性倾向"（如：看起来温暖→火属性）
- 稀有度影响孵出精灵的品质

## 六、地图与流程

### 6.1 爬塔结构（继承STS）
```
第一层: 普通敌人为主，1个Boss
第二层: 增加精英敌人，1个Boss
第三层: 精英+随机事件增多，最终Boss
```

### 6.2 房间类型
| 房间 | 功能 |
|------|------|
| ⚔️ 战斗 | 普通敌人战斗 |
| 👹 精英 | 强敌战斗，高奖励 |
| 🔥 篝火 | 选择：回复50%HP / 孵化蛋 |
| 🏪 商店 | 购买精灵球、装备、蛋 |
| ❓ 事件 | 随机剧情事件 |
| 👤 Boss | 楼层Boss |
| 🎲 宝藏 | 获得金币/装备/稀有道具 |

### 6.3 事件示例

#### 野生精灵遭遇
> 你遇到了一只受伤的[火焰狐]！
- [尝试捕获] 使用精灵球
- [战斗] 进入战斗
- [离开] 放弃机会

#### 神秘商人
> 一个裹着斗篷的商人拦住了你...
- [购买蛋] 300金币购买神秘蛋
- [出售精灵] 放弃一只精灵换取金币
- [离开]

#### 篝火传说
> 篝火旁坐着一位老者...
- [聆听故事] 随机获得知识（显示下一场Boss信息）
- [休息] 正常回复

## 八、技术实现要点

### 8.1 数据结构

#### 精灵数据
```typescript
interface Spirit {
  id: string;
  name: string;
  element: ElementType;
  rarity: Rarity;
  stats: {
    maxHp: number;
    atk: number;
    def: number;
    spd: number;
  };
  level: number;
  exp: number;
  skillDeck: Card[];      // 当前牌库
  learnableSkills: Card[]; // 可学习技能池
  equipment?: Equipment;
  status: StatusEffect[];
}
```

#### 卡牌数据
```typescript
interface Card {
  id: string;
  name: string;
  type: CardType;      // ATTACK / SKILL / POWER
  cost: number;
  element?: ElementType;
  effects: Effect[];
  owner: string;       // 所属精灵ID
}
```

### 8.2 关键算法

#### 伤害计算
```
damage = (baseDamage + ATK * 0.5) * elementMultiplier * modifier - DEF * 0.3
```

#### 抽牌逻辑
```
1. 收集所有存活精灵的skillDeck
2. 合并成masterDeck
3. 洗牌
4. 抽5张（+SPD加成）
```

## 九、视觉与交互设计

### 9.1 战斗界面布局
```
┌─────────────────────────────────────┐
│  [敌人1]  [敌人2]  [敌人3]           │ ← 敌方区域
│                                     │
│  ┌──────────┐    ┌──────────┐      │
│  │ 前排精灵  │    │ 后排精灵  │      │ ← 我方区域
│  └──────────┘    └──────────┘      │
│                                     │
│  [手牌区域 - 5张牌]                  │
│                                     │
│  [能量: 3/3]  [结束回合]  [捕获]    │ ← 操作区
└─────────────────────────────────────┘
```

### 9.2 关键动效
- 精灵切换：滑动过渡 + 粒子效果
- 技能释放：属性对应颜色特效
- 捕获成功：精灵球摇晃3次 + 闪光
- 升级：光芒环绕 + 新卡展示
