# DS01 - 深渊扫雷

**项目代号**: DS01  
**类型**: 网页游戏原型  
**核心玩法**: 扫雷 + 搜打撤 + 克苏鲁

## 游戏概述

将经典的扫雷玩法与"搜打撤"（搜索-战斗-撤离）机制相结合，融入克苏鲁式的疯狂元素。玩家需要在充满未知的网格中探索、收集物品、管理理智，并最终安全撤离。

## 核心机制

### 1. 扫雷基础
- 12x12 网格
- 20个隐藏陷阱（地雷）
- 数字提示周围陷阱数量
- 左右键标记可疑区域

### 2. 搜打撤系统
- **物品收集**: 化石、古老手稿、诅咒 idol、深渊遗物等
- **负重管理**: 有限背包空间，需要取舍
- **撤离点**: 随机生成在地图深处，到达后可选择撤离或继续探索
- **风险收益**: 越深风险越大，但奖励也更丰厚

### 3. 克苏鲁/疯狂元素
- **理智值系统**: 每次探索、触发陷阱、拾取诅咒物品都会降低理智
- **疯狂等级**: 
  - 100-70: 正常
  - 70-40: 轻微疯狂（偶尔听到低语）
  - 40-20: 中度疯狂（视觉扭曲，可看到隐藏物品轮廓）
  - 20-0: 重度疯狂（严重幻觉，可能看到虚假数字）
- **古神低语**: 随机触发的神秘文字提示
- **疯狂事件**: 遭遇不可名状的恐怖

## 操作说明

- **左键点击**: 揭示格子
- **右键点击**: 标记/取消标记
- **扫描**: 消耗理智，获得周围地雷信息
- **休息**: 缓慢恢复理智
- **撤离**: 到达撤离点后点击，结算分数

## 技术栈

- HTML5
- CSS3 (Grid, Flexbox, CSS变量)
- 原生 JavaScript (ES6+)

无需构建工具，直接用浏览器打开 `index.html` 即可游玩。

## 设计思路

这个项目尝试回答一个问题：**如何将三种看似不相关的游戏类型融合得自然且有深度？**

- 扫雷提供了**策略基础**和风险决策
- 搜打撤增加了**资源管理**和**撤离压力**
- 克苏鲁元素带来了**叙事氛围**和**不确定性**

三者结合创造了一种"平静的疯狂"——表面的网格逻辑下潜藏着不可名状的恐惧。
